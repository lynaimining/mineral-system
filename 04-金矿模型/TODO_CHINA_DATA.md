## P0：最高优先级（有渠道立即做）

### 1. Wind金融终端 API（WindPy）
**价值：** 最完整的中国矿业并购数据库，含资源量+交易金额
**需要：** Wind机构账号（约2-5万/年）
**操作：**
```python
import WindPy as w
w.start()
# 查询矿业并购重组数据
data = w.wset("mergeracquisition",
    "startdate=2018-01-01;enddate=2024-12-31;industry=采矿业;keyword=黄金")
# 或查询具体公司公告
ann = w.wset("anncategory",
    "windcode=600547.SH;startdate=2018-01-01;enddate=2024-12-31;announcementtype=重大资产重组")
```
**目标：** 找到20+笔国内金矿并购，提取资源量+交易金额→计算元/克

### 4. 各省自然资源厅矿权出让公告
**价值：** 政府挂牌出让有成交价，数据真实
**需要：** 人工访问各省网站
**重点省份：**
- 山东：http://dnr.shandong.gov.cn → 矿业权管理 → 出让结果公示
- 河南：http://zrzy.henan.gov.cn → 矿权出让
- 云南：http://zrzy.yunnan.gov.cn
- 贵州：http://zrzy.guizhou.gov.cn
- 内蒙古：http://zrzy.nmg.gov.cn

**注意：** 政府出让价格通常低于市场价（只含出让收益，不含溢价），需要说明

**操作：** 访问 http://mine168.com，搜索"黄金 采矿权"，记录挂牌价格和资源量
