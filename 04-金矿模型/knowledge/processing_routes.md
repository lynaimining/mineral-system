## 一、氰化浸出工艺

### 1.1 CIL（Carbon-In-Leach，炭浸法）

**工艺原理**：
- 浸出与吸附同时进行，活性炭直接加入氰化浸出矿浆中
- 金溶解后立即被活性炭吸附，减少"贵液损失"
- 适用于含泥高、有机碳高的矿石（防止"劫金"现象）

**典型流程**：
```
破碎（-10mm）→ 磨矿（80% -75μm）→ 浸出槽（6-8个串联，加NaCN + 活性炭）
→ 炭筛分离 → 载金炭解吸（高温高压，NaOH + NaCN溶液）
→ 电积（阴极沉积金泥）→ 熔炼（金锭，纯度 >99.5%）
```

**关键参数**：
- NaCN 浓度：0.03-0.05%（300-500 ppm）
- pH 值：10.5-11.5（加石灰调节）
- 浸出时间：24-48 小时
- 炭浓度：20-40 g/L
- 解吸温度：120-150°C，压力 200-400 kPa

**经济指标**：
- 金回收率：90-95%（自由金为主的矿石）
- 处理成本：$8-15/t（取决于磨矿细度和氰化物消耗）
- 资本投入：$15-25M（1000 t/d 规模）
- 氰化物消耗：0.3-0.8 kg NaCN/t 矿石

**典型应用**：
- 南非 Witwatersrand 金矿（Anglo American, Gold Fields）
- 澳大利亚 Kalgoorlie Super Pit（Northern Star Resources）
- 中国山东招远金矿群

---

### 1.2 CIP（Carbon-In-Pulp，炭浆法）

**工艺原理**：
- 浸出与吸附分步进行，先氰化浸出，再加活性炭吸附
- 适用于含泥低、矿石性质简单的矿石
- 流程比 CIL 简单，但贵液损失风险稍高

**典型流程**：
```
破碎 → 磨矿 → 浸出槽（氰化浸出，不加炭）→ 吸附槽（加活性炭）
→ 炭筛分离 → 解吸电积 → 熔炼
```

**关键参数**：
- 浸出时间：12-24 小时
- 吸附时间：6-12 小时
- 其他参数与 CIL 类似

**经济指标**：
- 金回收率：88-93%
- 处理成本：$7-12/t（比 CIL 略低）
- 适用于品位 >2 g/t 的简单矿石

---

### 1.3 堆浸（Heap Leaching）

**工艺原理**：
- 低成本、大规模处理低品位矿石的方法
- 矿石堆放在防渗垫上，喷淋氰化溶液，贵液从底部收集
- 适用于品位 0.5-2 g/t 的氧化矿或风化矿

**典型流程**：
```
爆破 → 粗碎（-150mm）→ 堆场（高度 5-10m）→ 喷淋 NaCN 溶液（50-150 天）
→ 贵液收集池 → Merrill-Crowe 沉淀法或活性炭吸附 → 熔炼
```

**关键参数**：
- 矿石粒度：-150mm 或 -50mm（破碎堆浸）
- NaCN 浓度：0.01-0.03%（100-300 ppm）
- 喷淋强度：5-15 L/h/m²
- 浸出周期：50-300 天（取决于矿石性质）

**经济指标**：
- 金回收率：50-75%（氧化矿）、30-50%（硫化矿）
- 处理成本：$3-8/t（极低）
- 资本投入：$5-10M（10,000 t/d 规模）
- 盈亏平衡品位：0.3-0.5 g/t

**典型应用**：
- 美国内华达州 Carlin Trend（Barrick Gold, Nevada Gold Mines）
- 秘鲁 Yanacocha 金矿（Newmont）
- 中国新疆阿希金矿（紫金矿业）

---

## 二、难处理金矿预处理

### 2.1 焙烧法（Roasting）

**适用矿石**：
- 含砷黄铁矿包裹金（Carlin 型、造山型部分矿床）
- 含碳质页岩的"劫金"矿石
- 硫化物含量 >5% 的难浸矿石

**工艺原理**：
- 高温（500-700°C）氧化焙烧，破坏硫化物和有机碳结构
- 金从包裹体中释放，转化为可氰化浸出的形态
- 同时氧化砷、硫，减少氰化物消耗

**典型流程**：
```
破碎 → 磨矿（80% -75μm）→ 流态化焙烧炉（600-650°C，停留时间 1-2h）
→ 焙砂冷却 → 氰化浸出（CIL/CIP）→ 尾气处理（脱硫、脱砷）
```

**关键参数**：
- 焙烧温度：600-650°C（过高会烧结，过低氧化不完全）
- 氧气浓度：15-21%
- 停留时间：1-2 小时
- 焙砂氧化率：>90%

**经济指标**：
- 金回收率提升：从 30-50% 提升至 85-92%
- 预处理成本：$15-25/t
- 资本投入：$30-50M（1000 t/d 规模）
- 环保成本：$5-10/t（尾气处理）

**典型应用**：
- 美国内华达州 Goldstrike（Barrick Gold，世界最大焙烧厂）
- 中国贵州烂泥沟金矿（紫金矿业）

**环保挑战**：
- SO₂ 和 As₂O₃ 排放，需严格尾气处理
- 焙砂含砷，需稳定化处理后堆存

---

### 2.2 生物氧化（Biooxidation）

**适用矿石**：
- 含砷黄铁矿包裹金（与焙烧法竞争）
- 硫化物含量 3-15% 的难浸矿石
- 环保要求严格、不允许焙烧的地区

**工艺原理**：
- 利用嗜酸性细菌（Acidithiobacillus ferrooxidans, A. thiooxidans）
- 在酸性环境（pH 1.5-2.5）下氧化硫化物，释放包裹金
- 反应温度低（40-50°C），环保友好

**典型流程**：
```
破碎 → 磨矿（80% -150μm）→ 生物氧化槽（5-7 天，加细菌、H₂SO₄、营养盐）
→ 固液分离 → 中和（加石灰至 pH 10.5）→ 氰化浸出（CIL）
```

**关键参数**：
- pH 值：1.5-2.5（硫酸调节）
- 温度：40-50°C（中温菌）或 65-75°C（高温菌）
- 停留时间：5-7 天
- 氧化率：>85%

**经济指标**：
- 金回收率提升：从 30-50% 提升至 80-88%
- 预处理成本：$10-18/t（比焙烧低）
- 资本投入：$25-40M（1000 t/d 规模）
- 运营成本：细菌培养、营养盐、温控

**典型应用**：
- 加纳 Ashanti Goldfields（Anglo Gold Ashanti）
- 澳大利亚 Wiluna 金矿（Wiluna Mining）
- 巴西 São Bento 金矿

**优势**：
- 环保友好，无 SO₂/As₂O₃ 排放
- 能耗低（不需高温）
- 适合中小规模矿山

**劣势**：
- 处理周期长（5-7 天 vs 焙烧 1-2 小时）
- 对矿石性质敏感（含铜、锌会抑制细菌）
- 寒冷地区需保温设施

---

### 2.3 压力氧化（POX, Pressure Oxidation）

**适用矿石**：
- 高硫化物含量（>15%）的难浸矿石
- 含砷黄铁矿包裹金
- 品位高（>5 g/t）、规模大的项目（经济性要求）

**工艺原理**：
- 高温（180-225°C）、高压（2-3 MPa）条件下，用氧气氧化硫化物
- 反应速度快（1-2 小时），氧化彻底
- 适合处理焙烧和生物氧化都难以处理的矿石

**典型流程**：
```
破碎 → 磨矿（80% -75μm）→ 高压釜（200°C, 2.5 MPa, 1-2h, 加 O₂）
→ 闪蒸冷却 → 固液分离 → 中和 → 氰化浸出（CIL）
```

**关键参数**：
- 温度：180-225°C
- 压力：2-3 MPa
- 氧分压：0.5-1.0 MPa
- 停留时间：1-2 小时
- 矿浆浓度：30-40% 固体

**经济指标**：
- 金回收率：88-95%（最高）
- 预处理成本：$20-35/t（最高）
- 资本投入：$50-80M（1000 t/d 规模，最高）
- 适用于品位 >5 g/t 的高品位矿石

**典型应用**：
- 加拿大 Campbell Mine（Goldcorp，已关闭）
- 俄罗斯 Olimpiada 金矿（Polyus Gold）
- 乌兹别克斯坦 Muruntau 金矿（Navoi Mining）

**优势**：
- 处理速度快，氧化彻底
- 金回收率最高
- 适合高硫、高砷、高碳的"三高"矿石

**劣势**：
- 资本投入和运营成本最高
- 设备复杂，维护要求高
- 仅适合大规模、高品位项目

---

### 2.4 超细磨（Ultrafine Grinding）

**适用矿石**：
- 金粒度极细（<10 μm）的难浸矿石
- 不含砷、硫化物含量低、但金被硅酸盐包裹的矿石
- 作为预处理的补充手段

**工艺原理**：
- 将矿石磨至 P₈₀ = 10-20 μm（常规磨矿 P₈₀ = 75 μm）
- 破坏包裹体，增加金与氰化物的接触面积
- 不改变矿物化学性质，纯物理方法

**典型流程**：
```
常规磨矿（P₈₀ = 75 μm）→ 超细磨（搅拌磨、IsaMill, P₈₀ = 15 μm）
→ 氰化浸出（CIL，浸出时间可缩短至 12-18h）
```

**关键参数**：
- 目标粒度：P₈₀ = 10-20 μm
- 磨矿介质：陶瓷球或钢球（直径 2-6 mm）
- 能耗：50-80 kWh/t（高）

**经济指标**：
- 金回收率提升：5-15%（取决于原矿性质）
- 额外成本：$5-12/t
- 适用于品位 >3 g/t 的矿石

**典型应用**：
- 澳大利亚 Mt Rawdon 金矿（Evolution Mining）
- 加拿大 Red Lake 金矿（Evolution Mining）

---

## 三、卡林型微细粒金处理

### 3.1 技术难点

**矿石特征**：
- 金粒度极细（<1 μm），以"不可见金"形式存在于黄铁矿、砷黄铁矿晶格中
- 含砷高（As 0.1-5%），含有机碳（C 0.1-3%）
- 矿物组成复杂：黄铁矿、砷黄铁矿、雄黄、雌黄、碳质页岩

**浸出障碍**：
- 金被硫化物包裹，氰化物无法接触
- 有机碳"劫金"现象严重（吸附已溶解的金）
- 砷矿物消耗大量氰化物和氧气

---

### 3.2 解决方案

**方案 1：焙烧 + 氰化**（主流）
- 600-650°C 焙烧，破坏硫化物和有机碳
- 金回收率：85-92%
- 典型应用：Barrick Goldstrike（世界最大卡林型金矿）

**方案 2：压力氧化 + 氰化**
- 200°C, 2.5 MPa 压力氧化
- 金回收率：88-95%
- 成本高，仅适合高品位矿石

**方案 3：超细磨 + 强化氰化**
- 磨至 P₈₀ = 10-15 μm
- 提高氰化物浓度至 0.1-0.15%
- 金回收率：70-80%（低于预处理方案）
- 适合小规模、环保要求严格的项目

---

### 3.3 典型案例

**Goldstrike Mine（美国内华达州）**：
- 运营商：Barrick Gold
- 处理能力：10,000 t/d（焙烧）+ 20,000 t/d（堆浸）
- 焙烧工艺：流态化焙烧炉，650°C
- 金回收率：90%（焙烧矿石）、65%（堆浸矿石）
- 年产金：~1.5 Moz（45 吨）

**Cortez Hills（美国内华达州）**：
- 运营商：Nevada Gold Mines（Barrick + Newmont JV）
- 处理能力：8,000 t/d
- 工艺：压力氧化 + CIL
- 金回收率：92%
- 平均品位：5.2 g/t

## 四、重选工艺

### 4.1 适用矿石

**理想条件**：
- 金粒度粗（>100 μm），比重大（19.3 g/cm³）
- 脉石矿物比重小（石英 2.65 g/cm³）
- 金呈游离状态，不被硫化物包裹

**典型矿床类型**：
- 砂金矿（冲积型、残坡积型）
- 石英脉型金矿（粗粒金为主）
- 部分造山型金矿（自由金含量高）

---

### 4.2 工艺流程

**典型流程**：
```
破碎（-10mm）→ 跳汰机（粗选，回收 >1mm 粗粒金）
→ 摇床（精选，回收 0.1-1mm 中粒金）
→ 离心选矿机（回收 0.02-0.1mm 细粒金）
→ 重选尾矿 → 氰化浸出（回收微细粒金）
```

**关键设备**：
- 跳汰机：处理能力大，回收粗粒金
- 摇床：精度高，适合中粒金
- 离心选矿机（Knelson, Falcon）：回收细粒金，金回收率 >95%

---

### 4.3 经济性

**经济指标**：
- 金回收率：60-80%（仅重选）、85-92%（重选 + 氰化）
- 处理成本：$3-6/t（极低）
- 资本投入：$5-10M（1000 t/d 规模）
- 适用于品位 >2 g/t、粗粒金含量 >50% 的矿石

**优势**：
- 成本低，无化学药剂
- 环保友好，无氰化物污染
- 可快速回收粗粒金，改善现金流

**劣势**：
- 无法回收微细粒金（<20 μm）
- 金回收率低于氰化法
- 需与氰化法组合使用

**典型应用**：
- 中国山东招远金矿（重选 + CIL）
- 俄罗斯 Sukhoi Log 金矿（重选预富集 + 氰化）
- 加纳小型砂金矿

---

## 五、浮选工艺

### 5.1 适用矿石

**理想条件**：
- 金与硫化物共生（黄铁矿、砷黄铁矿、黄铜矿）
- 金粒度细（<50 μm），难以重选
- 硫化物含量 >3%

**典型矿床类型**：
- 造山型金矿（金-黄铁矿共生）
- 浅成低温热液型（金-黄铁矿-石英脉）
- 斑岩型金铜矿（金-黄铜矿共生）

---

### 5.2 工艺流程

**典型流程**：
```
破碎 → 磨矿（80% -75μm）→ 浮选（加捕收剂、起泡剂）
→ 金精矿（Au 30-100 g/t，富集比 10-30 倍）
→ 精矿焙烧或压力氧化 → 氰化浸出
```

**关键参数**：
- 捕收剂：黄药（Xanthate）、黑药（Dithiophosphate）
- 起泡剂：松油、MIBC
- pH 值：8-10（石灰调节）
- 浮选时间：5-10 分钟

---

### 5.3 与氰化的组合

**方案 1：浮选 + 精矿氰化**
- 浮选富集金至精矿（富集比 10-30 倍）
- 精矿氰化浸出，尾矿堆存
- 优势：减少氰化处理量，降低成本
- 适用于低品位、大规模矿山

**方案 2：浮选 + 精矿焙烧 + 氰化**
- 浮选富集金和硫化物
- 精矿焙烧，破坏硫化物包裹
- 焙砂氰化浸出
- 适用于难处理金矿

**经济指标**：
- 金回收率：85-92%（浮选 + 氰化）
- 处理成本：$10-18/t
- 资本投入：$20-35M（1000 t/d 规模）

**典型应用**：
- 加拿大 Red Lake 金矿（浮选 + 压力氧化 + CIL）
- 中国紫金山金矿（浮选 + 焙烧 + CIL）

---

## 六、含砷金矿环保处理

### 6.1 砷的危害

**环境风险**：
- 砷是剧毒元素，饮用水标准 <10 μg/L（WHO）
- 焙烧产生 As₂O₃ 气体，需严格尾气处理
- 焙砂、尾矿含砷，需稳定化处理后堆存
- 氰化尾液含砷，需处理达标后排放

**健康风险**：
- 长期接触砷会导致皮肤癌、肺癌、肝损伤
- 矿山工人需严格防护

---

### 6.2 处理技术

**焙烧尾气处理**：
```
焙烧炉 → 旋风除尘器 → 电除尘器 → 湿法洗涤塔（NaOH 溶液吸收 As₂O₃）
→ 砷渣稳定化（加 FeSO₄ 生成 FeAsO₄ 沉淀）→ 安全堆存
```

**焙砂稳定化**：
- 加石灰、水泥固化，降低砷浸出率
- 堆存于防渗尾矿库，定期监测地下水

**氰化尾液处理**：
- SO₂/Air 法：破坏氰化物，沉淀砷
- 生物处理：利用细菌降解氰化物
- 达标排放：As <0.5 mg/L, CN <0.5 mg/L

---

### 6.3 环保合规

**中国标准**：
- 《黄金行业污染物排放标准》（GB 25466-2010）
- 焙烧尾气：As <0.4 mg/m³
- 废水：As <0.5 mg/L, CN <0.5 mg/L
- 尾矿库：防渗系数 <10⁻⁷ cm/s

**国际标准**：
- IFC Performance Standards（世界银行）
- 要求环境影响评估（EIA）、社区咨询、长期监测

**典型案例**：
- Barrick Goldstrike：投资 $200M 建设尾气处理系统，As 排放 <0.1 mg/m³
- 中国贵州烂泥沟金矿：焙砂水泥固化，尾矿库双层防渗

## 七、不同类型金矿的回收率对比

### 7.1 造山型金矿

**矿石特征**：
- 金粒度：10-100 μm（中等）
- 硫化物含量：3-10%（黄铁矿为主）
- 有机碳含量：低（<1%）

**推荐工艺**：
- 简单矿石：CIL/CIP，回收率 90-95%
- 难处理矿石：浮选 + 焙烧 + CIL，回收率 85-92%

**典型矿山**：
- 加拿大 Kirkland Lake（CIL，回收率 93%）
- 澳大利亚 Kalgoorlie（CIL，回收率 91%）

---

### 7.2 卡林型金矿

**矿石特征**：
- 金粒度：<1 μm（极细）
- 硫化物含量：5-15%（砷黄铁矿为主）
- 有机碳含量：高（1-3%）

**推荐工艺**：
- 氧化矿：堆浸，回收率 60-75%
- 硫化矿：焙烧 + CIL，回收率 85-92%
- 高品位矿：压力氧化 + CIL，回收率 88-95%

**典型矿山**：
- Barrick Goldstrike（焙烧 + CIL，回收率 90%）
- Newmont Carlin Trend（堆浸，回收率 70%）

---

### 7.3 浅成低温热液型

**矿石特征**：
- 金粒度：1-50 μm（细-中等）
- 硫化物含量：1-5%（黄铁矿、黄铜矿）
- 氧化程度：高（表层）至低（深部）

**推荐工艺**：
- 氧化矿：CIL/CIP，回收率 88-93%
- 硫化矿：浮选 + CIL，回收率 85-90%

**典型矿山**：
- 印尼 Grasberg（CIL，回收率 89%）
- 菲律宾 Lepanto（浮选 + CIL，回收率 87%）

---

### 7.4 斑岩型金铜矿

**矿石特征**：
- 金粒度：5-50 μm（细-中等）
- 硫化物含量：>10%（黄铜矿、辉钼矿）
- 金品位：低（0.3-1 g/t），铜品位：0.3-0.8%

**推荐工艺**：
- 浮选富集铜金精矿 → 铜冶炼回收金（副产品）
- 或：浮选尾矿堆浸回收金

**典型矿山**：
- 智利 Chuquicamata（浮选 + 冶炼，金回收率 75%）
- 美国 Bingham Canyon（浮选 + 冶炼，金回收率 70%）

## 八、2026 年技术趋势

### 8.1 无氰提金技术

**背景**：
- 氰化物剧毒，环保压力大
- 欧盟部分国家禁止新建氰化金矿
- 寻找环保替代技术成为行业热点

**主要技术**：

**1. 硫代硫酸盐浸出**：
- 反应：Au + 5S₂O₃²⁻ + O₂ → Au(S₂O₃)₂³⁻
- 优势：无毒，pH 中性，浸出速度快
- 劣势：药剂消耗高（$15-25/t），铜离子干扰严重
- 应用：Barrick Goldstrike 试验（2024-2025）

**2. 硫脲浸出**：
- 反应：Au + 2CS(NH₂)₂ + Fe³⁺ → Au[CS(NH₂)₂]₂⁺
- 优势：浸出速度快（比氰化快 10 倍）
- 劣势：药剂成本高，稳定性差
- 应用：小规模试验，尚未工业化

**3. 卤素浸出（氯化、溴化）**：
- 反应：2Au + 3Cl₂ + 2Cl⁻ → 2AuCl₄⁻
- 优势：浸出速度快，适合难处理矿石
- 劣势：腐蚀性强，设备要求高
- 应用：澳大利亚 CSIRO 研发中

**现状**：
- 氰化法仍是主流（>90% 金矿使用）
- 无氰技术成本高，尚未大规模应用
- 预计 2030 年后逐步推广

---

### 8.2 纳米气泡浮选

**技术原理**：
- 产生直径 <1 μm 的纳米气泡
- 增加气泡与矿粒的接触面积，提高浮选效率
- 特别适合微细粒金矿（<10 μm）

**优势**：
- 金回收率提升 5-10%
- 浮选时间缩短 30-50%
- 药剂消耗降低 20%

**应用**：
- 澳大利亚 Newcrest Mining 试验（2025）
- 中国山东黄金集团试验（2024-2025）

**挑战**：
- 设备成本高（纳米气泡发生器）
- 工艺参数优化复杂

---

### 8.3 智能选矿

**技术内容**：
- 在线矿石品位检测（XRF、LIBS）
- AI 优化浮选/氰化参数
- 数字孪生技术模拟选矿流程

**优势**：
- 金回收率提升 2-5%
- 药剂消耗降低 10-15%
- 能耗降低 5-10%

**应用**：
- Barrick Gold：AI 优化 Goldstrike 焙烧炉参数（2024）
- Newmont：数字孪生技术应用于 Boddington 选矿厂（2025）
- 中国紫金矿业：智能选矿系统应用于紫金山金矿（2025）

│     └─ 浮选 + 焙烧 + CIL → 回收率 85-92%

## 十、项目评估使用指南

### 10.1 工艺选择评估

**步骤 1：矿石性质分析**
- 金粒度分布（筛析 + 显微镜）
- 硫化物含量（化学分析）
- 砷含量（ICP-MS）
- 有机碳含量（燃烧法）
- 氧化率（硫酸铜浸出试验）

**步骤 2：小型试验**
- 氰化浸出试验（瓶滚试验，24-48h）
- 重选试验（摇床试验）
- 浮选试验（浮选柱试验）
- 预处理试验（焙烧/生物氧化/压力氧化）

**步骤 3：工艺方案对比**
- 金回收率
- 处理成本（$/t）
- 资本投入（$/t/d）
- 环保合规性
- 技术成熟度

---

### 10.2 回收率估算

**基于矿石类型的经验值**：
- 造山型（简单）：90-95%（CIL）
- 造山型（难处理）：85-92%（预处理 + CIL）
- 卡林型（氧化）：60-75%（堆浸）
- 卡林型（硫化）：85-92%（焙烧 + CIL）
- 浅成低温热液：88-93%（CIL）
- 斑岩型金铜：70-80%（浮选 + 冶炼）

**修正因子**：
- 金粒度 <5 μm：-5% 至 -10%
- 有机碳 >2%：-5% 至 -8%
- 砷含量 >1%：-3% 至 -5%
- 铜含量 >0.5%：-2% 至 -5%（干扰氰化）

---

### 10.3 成本估算

**处理成本构成**：
- 磨矿：$3-6/t（能耗为主）
- 氰化药剂：$1-3/t（NaCN $2-4/kg）
- 活性炭：$0.5-1/t
- 预处理（如需）：$10-25/t
- 尾矿处理：$1-3/t
- 人工 + 维护：$2-5/t

**总成本估算**：
- CIL/CIP：$8-15/t
- 堆浸：$3-8/t
- 浮选 + CIL：$15-25/t
- 焙烧 + CIL：$25-35/t
- 压力氧化 + CIL：$30-45/t

**资本投入估算**：
- CIL/CIP：$15,000-25,000 per t/d
- 堆浸：$500-1,000 per t/d
- 焙烧厂：$30,000-50,000 per t/d
- 压力氧化：$50,000-80,000 per t/d
