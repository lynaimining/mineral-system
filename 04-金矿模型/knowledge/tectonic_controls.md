# 金矿构造控矿规律

> 本文档总结各类型金矿的构造控矿规律，用于成矿潜力预测和勘探靶区优选。

---

## 一、造山型金矿构造控矿规律

### 1.1 构造背景

**成矿构造环境**：
- 克拉通边缘造山带
- 绿岩带剪切带系统
- 逆冲-走滑断裂系统

**成矿时代**：
- 太古宙：2.6-2.7 Ga（Yilgarn、Abitibi）
- 古元古代：2.0-2.2 Ga（西非Birimian）
- 显生宙：各造山期均有

### 1.2 控矿构造类型

#### 1.2.1 剪切带控矿

**特征**：
- 区域性剪切带（长度10-100 km）
- 次级剪切带（长度1-10 km）
- 矿体沿剪切带分布

**典型矿床**：
- Kalgoorlie（澳大利亚）：Boulder-Lefroy剪切带
- Kirkland Lake（加拿大）：Larder Lake-Cadillac剪切带
- Obuasi（加纳）：Ashanti剪切带

**控矿要素**：
- 剪切带宽度：10-500 m
- 剪切带倾角：30-70°
- 剪切带走向：与区域构造一致

#### 1.2.2 褶皱控矿

**特征**：
- 矿体位于背斜或向斜转折端
- 褶皱轴部应力集中
- 石英脉充填

**典型矿床**：
- Bendigo（澳大利亚）：背斜轴部
- Ballarat（澳大利亚）：向斜转折端

#### 1.2.3 断裂交汇控矿

**特征**：
- 多组断裂交汇部位
- 应力释放空间
- 矿体富集

**典型矿床**：
- Jundee（澳大利亚）：NE向与NW向断裂交汇
- Granny Smith（澳大利亚）：多组断裂交汇

### 1.3 构造控矿模式

**模式1：剪切带型**
```
区域剪切带 → 次级剪切带 → 石英脉群 → 金矿化
```

**模式2：褶皱-剪切带复合型**
```
褶皱轴部 + 剪切带 → 应力集中 → 石英脉充填 → 金矿化
```

**模式3：断裂交汇型**
```
多组断裂交汇 → 应力释放 → 石英脉充填 → 金矿化
```

### 1.4 勘探指示

**关键构造要素**：
1. 区域性剪切带（长度>10 km）
2. 次级剪切带（宽度10-500 m）
3. 褶皱转折端
4. 断裂交汇部位

**勘探靶区优选**：
- 剪切带与绿岩带接触部位
- 剪切带宽度变化部位
- 断裂交汇部位
- 褶皱轴部

## 二、卡林型金矿构造控矿规律

### 2.1 构造背景

**成矿构造环境**：
- 伸展构造背景
- 碳酸盐岩容矿
- 低角度逆冲断层 + 高角度正断层

**成矿时代**：
- 始新世（42-36 Ma）

### 2.2 控矿构造类型

#### 2.2.1 低角度逆冲断层

**特征**：
- Roberts Mountains逆冲断层（Nevada）
- 断层倾角：10-30°
- 断层位移：数十公里

**控矿作用**：
- 提供流体通道
- 控制矿体分布

#### 2.2.2 高角度正断层

**特征**：
- 断层倾角：60-80°
- 断层走向：NE-NNE
- 断层间距：1-5 km

**控矿作用**：
- 流体上升通道
- 矿体直接赋存

**典型矿床**：
- Goldstrike：Post-Betze断层
- Cortez：Cortez断层

#### 2.2.3 岩性接触带

**特征**：
- 碳酸盐岩与碎屑岩接触
- 岩性界面控矿

**典型矿床**：
- Carlin：Roberts Mountains组与Popovich组接触

### 2.3 构造控矿模式

**模式：逆冲断层 + 正断层复合型**
```
低角度逆冲断层（区域构造） → 高角度正断层（流体通道） → 碳酸盐岩容矿 → 金矿化
```

### 2.4 勘探指示

**关键构造要素**：
1. Roberts Mountains逆冲断层
2. 高角度���断层（NE-NNE走向）
3. 碳酸盐岩与碎屑岩接触带

**勘探靶区优选**：
- 逆冲断层上盘碳酸盐岩
- 高角度正断层与碳酸盐岩交汇部位
- 岩性接触带

## 三、浅成低温热液型金矿构造控矿规律

### 3.1 构造背景

**成矿构造环境**：
- 火山弧伸展环境
- 环太平洋俯冲带
- 火山-构造活动区

**成矿时代**：
- 新生代为主（<50 Ma）

### 3.2 控矿构造类型

#### 3.2.1 火山口-火山穹隆

**特征**：
- 火山口周缘
- 火山穹隆顶部
- 环形断裂

**典型矿床**：
- Yanacocha（秘鲁）：火山穹隆顶部
- Pueblo Viejo（多米尼加）：火山口周缘

#### 3.2.2 断裂控矿

**高硫型（HS）**：
- 环形断裂
- 放射状断裂
- 断层倾角：60-80°

**低硫型（LS）**：
- 张性断裂
- 断层倾角：70-85°
- 断层走向：与区域构造一致

**典型矿床**：
- Hishikari（日本）：NW-SE向张性断裂
- Porgera（巴布亚新几内亚）：NE-SW向断裂

#### 3.2.3 岩脉控矿

**特征**：
- 岩脉与围岩接触带
- 岩脉内部裂隙

**典型矿床**：
- Martha Hill（新西兰）：安山岩脉

### 3.3 构造控矿模式

**高硫型（HS）模式**：
```
火山口/火山穹隆 → 环形/放射状断裂 → 热液上升 → 金矿化
```

**低硫型（LS）模式**：
```
张性断裂 → 热液充填 → 石英脉型金矿化
```

### 3.4 勘探指示

**关键构造要素**：
1. 火山口/火山穹隆
2. 环形/放射状断裂（HS）
3. 张性断裂（LS）
4. 岩脉

**勘探靶区优选**：
- 火山穹隆顶部
- 火山口周缘
- 断裂交汇部位
- 岩脉与围岩接触带

## 四、斑岩型金铜矿构造控矿规律

### 4.1 构造背景

**成矿构造环境**：
- 俯冲带岩浆弧
- 环太平洋俯冲带
- 碰撞造山带

**成矿时代**：
- 中生代-新生代为主

### 4.2 控矿构造类型

#### 4.2.1 岩体侵入构造

**特征**：
- 斑岩体侵入
- 岩体规模：0.5-5 km²
- 岩体形态：岩株、岩枝

**典型矿床**：
- Grasberg（印尼）：闪长岩-石英闪长岩岩体
- Cadia（澳大利亚）：石英二长岩岩体
- Oyu Tolgoi（蒙古）：石英闪长岩岩体

#### 4.2.2 断裂控矿

**特征**：
- 区域性断裂控制岩体侵入
- 断裂走向：与俯冲带平行
- 断裂倾角：60-80°

**控矿作用**：
- 控制岩浆上升
- 控制矿体分布

#### 4.2.3 网状裂隙

**特征**：
- 岩体内部网状裂隙
- 裂隙密度：10-50条/m
- 裂隙充填：石英-硫化物脉

**控矿作用**：
- 矿化直接赋存

### 4.3 构造控矿模式

**模式：岩体侵入 + 网状裂隙型**
```
俯冲带岩浆上升 → 斑岩体侵入 → 网状裂隙发育 → 热液充填 → 金铜矿化
```

### 4.4 勘探指示

**关键构造要素**：
1. 斑岩体（0.5-5 km²）
2. 区域性断裂
3. 网状裂隙

**勘探靶区优选**：
- 斑岩体中心
- 斑岩体与围岩接触带
- 区域性断裂与岩体交汇部位

## 五、IOCG型金矿构造控矿规律

### 5.1 构造背景

**成矿构造环境**：
- 克拉通边缘伸展环境
- 裂谷系统
- 深大断裂

**成矿时代**：
- 古元古代-中元古代为主（1.5-2.0 Ga）

### 5.2 控矿构造类型

#### 5.2.1 深大断裂

**特征**：
- 断裂规模：长度>100 km
- 断裂倾角：60-80°
- 断裂深度：切穿地壳

**典型矿床**：
- Olympic Dam（澳大利亚）：Olympic Dam断裂
- Carajás（巴西）：Carajás断裂

#### 5.2.2 角砾岩筒

**特征**：
- 角砾岩筒直径：100-1000 m
- 角砾岩筒深度：>1000 m
- 角砾岩筒形态：管状、筒状

**控矿作用**：
- 矿化直接赋存

**典型矿床**：
- Olympic Dam：角砾岩筒型
- Ernest Henry（澳大利亚）：角砾岩筒型

#### 5.2.3 剪切带

**特征**��
- 剪切带宽度：10-500 m
- 剪切带倾角：30-70°

**典型矿床**：
- Carajás：剪切带型

### 5.3 构造控矿模式

**模式1：深大断裂 + 角砾岩筒型**
```
深大断裂 → 流体上升 → 角砾岩筒形成 → 金铜矿化
```

**模式2：剪切带型**
```
剪切带 → 流体通道 → 金铜矿化
```

### 5.4 勘探指示

**关键构造要素**：
1. 深大断裂（长度>100 km）
2. 角砾岩筒
3. 剪切带

**勘探靶区优选**：
- 深大断裂与岩体交汇部位
- 角砾岩筒
- 剪切带与岩体接触带

## 六、Witwatersrand型金矿构造控矿规律

### 6.1 构造背景

**成矿构造环境**：
- 古元古代前陆盆地
- 克拉通边缘沉积盆地

**成矿时代**：
- 2.9-2.7 Ga

### 6.2 控矿构造类型

#### 6.2.1 盆地构造

**特征**：
- 盆地规模：300 × 200 km
- 盆地深度：>7000 m
- 盆地类型：前陆盆地

**控矿作用**：
- 控���砾岩层分布
- 控制金矿化分布

#### 6.2.2 古河道

**特征**：
- 古河道宽度：1-10 km
- 古河道深度：10-100 m
- 古河道走向：NE-SW

**控矿作用**：
- 控制砾岩层厚度
- 控制金品位

**典型矿床**：
- Witwatersrand Basin：多条古河道

#### 6.2.3 断裂控矿

**特征**：
- 逆冲断层
- 断层倾角：30-60°
- 断层位移：数百米至数公里

**控矿作用**：
- 控制矿体分布
- 后期改造矿体

### 6.3 构造控矿模式

**模式：盆地 + 古河道型**
```
前陆盆地 → 古河道沉积 → 砾岩层 → 金矿化 → 后期断裂改造
```

### 6.4 勘探指示

**关键构造要素**：
1. 盆地构造
2. 古河道
3. 砾岩层

**勘探靶区优选**：
- 古河道中心
- 砾岩层厚度大的部位
- 断裂与砾岩层交汇部位

## 七、构造控矿规律对比

### 7.1 七种类型对比

| 类型 | 主控构造 | 次要构造 | 构造尺度 | 成矿深度 |
|------|---------|---------|---------|---------|
| **造山型** | 剪切带 | 褶皱、断裂交汇 | 区域性（10-100 km） | 中-深（2-10 km） |
| **卡林型** | 逆冲断层 + 正断层 | 岩性接触带 | 区域性（10-50 km） | 浅（<1 km） |
| **浅成HS** | 火山口/火山穹隆 | 环形/放射状断裂 | 局部（1-5 km） | 浅（<1 km） |
| **浅成LS** | 张性断裂 | 岩脉 | 局部（1-10 km） | 浅-中（1-3 km） |
| **斑岩型** | 岩体侵入 | 网状裂隙 | 局部（0.5-5 km） | 中（2-5 km） |
| **IOCG型** | 深大断裂 | 角砾岩筒、剪切带 | 区域性（>100 km） | 深（>5 km） |
| **Witwatersrand** | 盆地构造 | 古河道、断裂 | 区域性（>100 km） | 深（>3 km） |

### 7.2 构造控矿关键要素

**区域性构造**：
- 造山型：剪切带系统
- 卡林型：逆冲断层系统
- IOCG型：深大断裂系统
- Witwatersrand型：盆地构造

**局部构造**：
- 浅成低温热液型：火山口/火山穹隆、断裂
- 斑岩型：岩体侵入、网状裂隙

## 八、勘探应用指南

### 8.1 构造解析流程

**Step 1**：区域构造背景分析
- 确定构造环境（造山带/伸展带/俯冲带/盆地）
- 识别区域性构造（剪切带/断裂/褶皱）

**Step 2**：局部构造解析
- 识别次级构造（断裂交汇/褶皱转折/岩体侵入）
- 分析构造控矿作用

**Step 3**：构造控矿模式判定
- 对比典型矿床构造模式
- 确定项目构造控矿模式

**Step 4**：勘探靶区优选
- 圈定有利构造部位
- 优选勘探靶区

### 8.2 构造控矿评分标准

**造山型金矿**：
- 区域性剪切带（30分）
- 次级剪切带（20分）
- 褶皱转折端（15分）
- 断裂交汇（15分）
- 绿岩带（20分）

**卡林型金矿**：
- 逆冲断层（30分）
- 高角度正断层（25分）
- 碳酸盐岩（25分）
- 岩性接触带（20分）

**浅成低温热液型金矿**：
- 火山口/火山穹隆（30分）
- 环形/放射状断裂（HS，25分）
- 张性断裂（LS，25分）
- 火山岩（20分）

**斑岩型金铜矿**：
- 斑岩体（40分）
- 网状裂隙（30分）
- 区域性断裂（20分）
- 蚀变分带（10分）

**IOCG型金矿**：
- 深大断裂（35分）
- 角砾岩筒（30分）
- 剪切带（20分）
- 铁氧化物蚀变（15分）

### 8.3 典型矿床构造控矿案例

**控矿构造**：

**控矿构造**：
- 主构造：Roberts Mountains逆冲断层

- 深度>1000 m
