## 报告结构（10章 + 参考文献 + 附录）

### 封面（1页）
- 标题：{项目名称} 金矿项目地质评价与投资建议报告
- 编制单位：凌云智矿 LynAI Mines
- 日期

### 执行摘要（2页）
- 项目概况（1段）
- 关键发现（5-7条bullet）
- 主要风险（5条bullet）
- 投资建议（分阶段策略，含预算框架）

- 3.1 矿区地层

- 3.5 矿石加工选冶特性（初步）
- 3.6 信息缺口
- **必须插入**：矿区地质图、矿体剖面图（从PPT/PDF提取）

## 图件要求（强制）

每份报告必须包含以下图件：
1. **区域地质图**（从项目材料提取）
2. **矿区地质图**（从项目材料提取）
3. **品位-吨位对比图**（matplotlib生成，含全球标杆矿床）
4. **成矿要素雷达图**（matplotlib生成，10项评分）
5. **风险矩阵图**（matplotlib生成，影响×可能性）
6. **其他项目材料图件**（从PPT/PDF提取的关键图片）

图件提取流程：
1. 用zipfile从PPTX/DOCX的media目录提取所有图片
2. 用PyMuPDF从PDF提取图片
3. 按文件大小排序，选择>500KB的图片（通常是地质图等关键图件）
4. 每张图必须有图注（图X-Y 具体描述）

## 质量标准

### 篇幅要求
- 最低：25页（材料极度有限时）
- 标准：30-40页（正常情况）
- 上限：50页（材料充分时）

### 内容深度
- 每个章节至少500字正文
- 每章至少1个数据表格
- 关键结论必须有数据支撑
- 所有推测性判断必须标注"推测"或"需验证"

### 信息缺口声明（强制）
- 每章末尾列出该章节的信息缺口
- 第9章汇总所有信息缺口（含获取方式和成本）
- 明确区分"确定的结论"和"推测性判断"

### 风险平衡（强制）
- 风险和有利因素必须平衡呈现
- 禁止全面乐观的"一边倒"结论
- 经济性分析必须含敏感性分析

### 数据来源标注（强制）
- 所有数据标注来源
- 过时数据标注"截至XX年"
- 假设条件明确列出

```python
# 品位-吨位对比图生成示例
import matplotlib.pyplot as plt
import matplotlib
matplotlib.rcParams['font.family'] = ['Microsoft YaHei']
matplotlib.rcParams['axes.unicode_minus'] = False

# 数据准备
benchmark_deposits = {
    'Kalgoorlie': {'tonnage': 1500, 'grade': 4.5, 'tier': 'Tier 0'},
    'Bendigo': {'tonnage': 300, 'grade': 5.2, 'tier': 'Tier 1'},
    'Val-d\'Or': {'tonnage': 250, 'grade': 4.8, 'tier': 'Tier 1'},
    'Yilgarn-A': {'tonnage': 80, 'grade': 3.5, 'tier': 'Tier 2'},
    'Abitibi-B': {'tonnage': 50, 'grade': 6.0, 'tier': 'Tier 2'},
}

project = {'tonnage': 项目金属量, 'grade': 项目品位}

# 绘图
fig, ax = plt.subplots(figsize=(10, 6))

# 绘制标杆矿床
for name, data in benchmark_deposits.items():
    color = 'red' if data['tier'] == 'Tier 0' else 'orange' if data['tier'] == 'Tier 1' else 'green'
    ax.scatter(data['tonnage'], data['grade'], s=100, c=color, alpha=0.6, edgecolors='black')
    ax.annotate(name, (data['tonnage'], data['grade']), xytext=(5, 5),
                textcoords='offset points', fontsize=9)

# 绘制项目
ax.scatter(project['tonnage'], project['grade'], s=200, c='blue', marker='*',
           edgecolors='black', linewidths=2, label='本项目', zorder=5)

# 设置
ax.set_xlabel('金属量（t Au）', fontsize=12)
ax.set_ylabel('品位（g/t Au）', fontsize=12)
ax.set_title('项目品位-吨位对比（造山型金矿）', fontsize=14, fontweight='bold')
ax.set_xscale('log')
ax.grid(True, alpha=0.3)
ax.legend()

plt.tight_layout()
plt.savefig('grade_tonnage_comparison.png', dpi=180, bbox_inches='tight')
plt.close()
```
