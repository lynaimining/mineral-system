## 一、项目目标（用大白话说就是要干什么）

**输入**：一张地质图（PDF/图片）
**输出**：
1. 最有可能成矿的靶区（标在地图上，按优先级排序）
2. 值得地质人员现场去看的具体位置（GPS坐标）
3. 现场应该重点关注什么现象（比如"找石英脉"、"看蚀变带"、"采样测金"）

**核心价值**：
- 把老专家20年经验变成AI，5分钟看完一张图
- 不漏掉任何可能的矿，也不浪费时间去没希望的地方
- 新手地质员也能干老手的活

## 二、技术路线（怎么实现）

### 2.1 基础架构

```
地质图 → 图像预处理 → Gemma 4视觉模型 → 特征提取 →
已知矿点数据库比对 → 成矿规律知识库匹配 → 靶区评分排序 →
输出报告（地图+坐标+现场指南）
```

### 2.2 核心技术组件

| 组件 | 技术方案 | 现有基础 |
|------|---------|---------|
| **图像预处理** | 地质图裁剪、图例识别、颜色标准化 | ✅ 有现成的 geo-map-crop skill（CIEDE2000色差算法） |
| **视觉理解** | Gemma 4 多模态模型（识别岩性、构造、蚀变） | ⚠️ 需要训练 |
| **知识库** | 7大矿种成矿规律（铜金镍锂+3种）| ✅ 有现成的 copper-ore/gold-ore/nickel-ore/lithium-ore skills |
| **空间比对** | 已知矿点数据库（经纬度+矿种+规模） | ⚠️ 需要建库 |
| **靶区评分** | 5要素打分（岩性25%+构造30%+蚀变20%+地化15%+地物10%）| ✅ 有现成的 geological-targeting skill |
| **报告生成** | 自动生成Word报告+标注地图 | ✅ 有现成的 docx/scientific-plot skills |

### 2.3 训练数据准备

**你现有的数据**：
- 几百张地质图 → 用于训练视觉模型识别岩性/构造/蚀变
- 一定数量的论文 → 提取成矿规律、典型矿床特征
- 一些JORC报告 → 提取已知矿点位置、品位、规模

**需要补充的数据**：
- 已知矿点数据库（全球主要矿床的GPS坐标+矿种+吨位）
  - 来源：USGS MRDS数据库（免费）、中国矿产地数据库、公司内部项目库
  - 预计规模：5000-10000个矿点
- 负样本数据（没有矿的地质图区域）
  - 用于训练模型不要"见啥都说有矿"

## 三、工作计划（分三个阶段）

### 阶段1：数据准备 + 基础模型训练（6周）

**Week 1-2：数据整理**
- [ ] 整理现有几百张地质图，按矿种分类（铜/金/镍/锂/其他）
- [ ] 从论文和JORC报告中提取已知矿点信息（位置+矿种+规模）
- [ ] 下载USGS MRDS数据库，建立全球矿点数据库
- [ ] 标注100张地质图的关键特征（岩性边界、断裂、蚀变带）

**Week 3-4：模型训练环境搭建**
- [ ] 租用云服务器（建议阿里云/腾讯云GPU实例）
- [ ] 部署Gemma 4模型（Google开源，可商用）
- [ ] 搭建训练pipeline（数据加载→预处理→训练→验证）
- [ ] 跑通第一个baseline模型（哪怕效果差也行）

**Week 5-6：第一轮训练**
- [ ] 训练Gemma 4识别地质图上的岩性单元（准确率目标：70%）
- [ ] 训练识别构造特征（断裂、褶皱、侵入接触）
- [ ] 训练识别蚀变特征（颜色异常、矿化蚀变带）
- [ ] 验证模型在测试集上的表现

**交付物**：
- 标注好的100张地质图数据集
- 全球矿点数据库（CSV格式，含经纬度）
- 第一版Gemma 4地质图理解模型（准确率≥70%）

---

### 阶段2：靶区识别算法开发（6周）

**Week 7-8：空间比对模块**
- [ ] 开发地质图坐标识别算法（从图例提取经纬度范围）
- [ ] 开发已知矿点叠加功能（在地图上标出周边已知矿）
- [ ] 开发相似性搜索（找出和已知大矿地质条件最像的区域）

**Week 9-10：成矿规律匹配**
- [ ] 整合现有7大矿种知识库（copper-ore等skills）
- [ ] 开发矿种自动识别（根据岩性+构造组合判断可能的矿种）
- [ ] 开发5要素评分算法（岩性/构造/蚀变/地化/地物）

**Week 11-12：靶区优选与排序**
- [ ] 开发靶区圈定算法（把高分区域圈成多边形）
- [ ] 开发优先级排序（综合评分+已知矿点距离+可达性）
- [ ] 开发现场指南生成（根据矿种和地质特征，自动生成"现场应该看什么"）

**交付物**：
- 靶区识别完整算法（从地质图到靶区坐标）
- 测试报告（在10张真实项目地质图上的识别效果）
- 第一版Web界面（上传地质图→显示靶区）

---

### 阶段3：系统集成 + 实战验证（6周）

**Week 13-14：系统集成**
- [ ] 开发完整Web应用（前端：上传地图+显示结果；后端：模型推理）
- [ ] 集成报告生成模块（自动生成Word报告+标注地图）
- [ ] 开发批量处理功能（一次上传多张地图，批量出结果）

**Week 15-16：实战验证**
- [ ] 选3-5个真实项目，用AI系统跑一遍
- [ ] 对比AI结果 vs 老专家结果（准确率、召回率、漏报率）
- [ ] 根据反馈调整模型参数和评分权重

**Week 17-18：优化与部署**
- [ ] 根据实战反馈优化模型（重点提升召回率，不能漏矿）
- [ ] 性能优化（单张地图处理时间<5分钟）
- [ ] 部署到生产服务器（或买自己的服务器）
- [ ] 编写用户手册和培训材料

**交付物**：
- 完整的Web应用（可以给客户用）
- 实战验证报告（准确率、召回率、案例对比）
- 用户手册和培训视频
