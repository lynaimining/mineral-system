## 一、项目目标（用大白话说就是要干什么）

### 1.1 核心功能

**输入**：一张地质图（PDF/图片）
**输出**：
1. 最有可能成矿的靶区（标在地图上，按优先级排序）
2. 值得地质人员现场去看的具体位置（GPS坐标）
3. 现场应该重点关注什么现象（比如"找石英脉"、"看蚀变带"、"采样测金"）

### 1.2 核心价值

**对内**：
- 把老专家20年经验变成AI，5分钟看完一张图
- 不漏掉任何可能的矿，也不浪费时间去没希望的地方
- 新手地质员也能干老手的活

**对外**：
- 新产品线：可以作为独立产品卖给勘探公司
- 提升竞争力：在矿业咨询投标中，"AI辅助靶区识别"是加分项
- 数据积累：每个项目的反馈都能优化模型，形成正循环

## 二、技术路线

### 2.1 基础架构

```
地质图 → 图像预处理 → Gemma 4视觉模型 → 特征提取 →
已知矿点数据库比对 → 成矿规律知识库匹配 → 靶区评分排序 →
输出报告（地图+坐标+现场指南）
```

### 2.2 核心技术组件

| 组件 | 技术方案 | 现有基础 |
|------|---------|---------|
| **图像预处理** | 地质图裁剪、图例识别、颜色标准化 | ✅ geo-map-crop skill（CIEDE2000色差算法） |
| **视觉理解** | Gemma 4 多模态模型（识别岩性、构造、蚀变） | ⚠️ 需要训练 |
| **知识库** | 金矿成矿规律（7大类型）| ✅ gold-ore skill |
| **空间比对** | 已知矿点数据库（经纬度+矿种+规模） | ⚠️ 需要建库 |
| **靶区评分** | 5要素打分（岩性25%+构造30%+蚀变20%+地化15%+地物10%）| ✅ geological-targeting skill |
| **报告生成** | 自动生成Word报告+标注地图 | ✅ docx/scientific-plot skills |

### 2.3 现有资产可复用

| 现有Skill | 可复用功能 | 节省成本 |
|-----------|-----------|---------|
| **geo-map-crop** | 地质图预处理、图例识别 | ¥20,000（省2周开发） |
| **geological-targeting** | 5要素评分算法、靶区圈定 | ¥30,000（省3周开发） |
| **gold-ore** | 金矿成矿规律知识库 | ¥50,000（省5周整理） |
| **docx/scientific-plot** | 报告生成、地图标注 | ¥15,000（省1.5周开发） |

**总计节省**：¥115,000 + 11.5周工作量

## 九、替代方案（如果Gemma 4不行）

### 方案B：相似性搜索（而非从零识别）

**思路**：
- 用户上传地质图 → AI在全球矿点数据库中找最相似的10个矿床 → 给出类比建议
- 基于检索（Retrieval）而非生成（Generation），技术难度低

**优势**：
- 不需要训练复杂的识别模型
- 结果可解释性强（"你的地质条件和澳大利亚XX矿床很像"）
- 开发周期短（1个月MVP）

**成本**：¥8-12万

### 方案C：产学研合作（而非自己招人）

**思路**：
- 找一个地质+AI交叉的高校团队合作
- 凌云提供数据和行业知识，高校提供AI人才和算力
- 做成科研项目，申请政府资助

**优势**：
- 人力成本降低80%
- 可能拿到政府资助（10-50万）
