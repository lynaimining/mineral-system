# 锂矿冶炼工艺路线对比

> 本文档对比硬岩型和盐湖型锂矿的冶炼工艺路线，用于项目经济评价和工艺选择。

---

## 一、硬岩型锂矿冶炼工艺

### 1.1 锂辉石精矿路线（主流）

**工艺流程**：
```
原矿 → 破碎 → 浮选 → 锂辉石精矿（6% Li₂O）→ 焙烧（1050-1100°C）→ 酸化 → 碳酸锂/氢氧化锂
```

**关键参数**：
- **精矿品位**：6-7% Li₂O
- **回收率**：75-85%
- **产品**：电池级碳酸锂（Li₂CO₃ ≥99.5%）或氢氧化锂（LiOH·H₂O ≥56.5%）

**经济参数**：
- **CAPEX**：$8,000-12,000/t LCE
- **OPEX**：$4,000-6,000/t LCE
- **建设周期**：2-3 年

**优势**：
- 技术成熟，全球主流工艺
- 可生产氢氧化锂（三元电池首选）
- 建设周期短

**劣势**：
- 能耗高（焙烧工序）
- 尾矿处理

---

### 1.2 锂云母路线

**工艺流程**：
```
原矿 → 破碎 → 浮选 → 锂云母精矿 → 硫酸法/石灰石焙烧法 → 碳酸锂
```

**关键参数**：
- **精矿品位**：3-5% Li₂O
- **回收率**：60-75%
- **产品**：碳酸锂

**经济参数**：
- **CAPEX**：$10,000-15,000/t LCE
- **OPEX**：$5,000-7,000/t LCE

**优势**：
- 适用于锂云母型矿床

**劣势**：
- 回收率低于锂辉石
- 成本较高

---

## 二、盐湖型锂矿冶炼工艺

### 2.1 传统蒸发法

**工艺流程**：
```
卤水抽取 → 蒸发池（18-24个月）→ 碳酸锂沉淀 → 精制 → 电池级碳酸锂
```

**关键参数**：
- **蒸发周期**：18-24 个月
- **回收率**：40-60%
- **适用条件**：Mg/Li < 10

**经济参数**：
- **CAPEX**：$5,000-8,000/t LCE
- **OPEX**：$2,000-4,000/t LCE
- **建设周期**：3-4 年

**优势**：
- 成本最低（Mg/Li < 10 时）
- 技术成熟

**劣势**：
- 周期长（18-24 个月）
- 占地面积大
- 水资源消耗
- Mg/Li > 10 时不经济

---

### 2.2 DLE（Direct Lithium Extraction）直接提锂

**工艺流程**：
```
卤水抽取 → 吸附/萃取 → 锂富集 → 碳酸锂沉淀 → 精制 → 电池级碳酸锂
```

**关键参数**：
- **提锂周期**：数周（vs 传统 18-24 个月）
- **回收率**：70-90%
- **适用条件**：Mg/Li > 10（优势明显）

**经济参数**：
- **CAPEX**：$8,000-12,000/t LCE
- **OPEX**：$4,000-6,000/t LCE
- **建设周期**：1-2 年

**优势**：
- 周期短（数周 vs 18-24 个月）
- 回收率高
- 占地面积小
- 适用于高 Mg/Li 比盐湖

**劣势**：
- 技术尚未完全成熟
- CAPEX 高于传统蒸发法

**技术提供商**：
- Lilac Solutions
- EnergyX
- Sunresin
- 蓝晓科技

## 三、产品等级对比

### 3.1 电池级 vs 工业级

| 产品 | 规格 | 应用 | 价格（2026） |
|------|------|------|------------|
| **电池级碳酸锂** | Li₂CO₃ ≥99.5% | 磷酸铁锂电池 | $15,000-25,000/t |
| **电池级氢氧化锂** | LiOH·H₂O ≥56.5% | 三元锂电池（NCM/NCA） | $18,000-28,000/t |
| **工业级碳酸锂** | Li₂CO₃ ≥99.0% | 玻璃/陶瓷 | $12,000-18,000/t |

**关键判断**：
- **氢氧化锂**：只能由硬岩型生产（盐湖型技术难度大）
- **价差**：电池级氢氧化锂 > 电池级碳酸锂 > 工业级碳酸锂

## 四、硬岩 vs 盐湖工艺对比

| 特征 | 硬岩型（锂辉石） | 盐湖型（传统蒸发） | 盐湖型（DLE） |
|------|----------------|------------------|--------------|
| **CAPEX** | 中（$8-12k/t） | 低（$5-8k/t） | 中（$8-12k/t） |
| **OPEX** | 中（$4-6k/t） | 低（$2-4k/t） | 中（$4-6k/t） |
| **建设周期** | 短（2-3年） | 长（3-4年） | 短（1-2年） |
| **生产周期** | 立即 | 长（18-24月） | 短（数周） |
| **回收率** | 高（75-85%） | 低（40-60%） | 高（70-90%） |
| **产品** | 碳酸锂+氢氧化锂 | 碳酸锂 | 碳酸锂 |
| **Mg/Li要求** | 无 | <10 | >10优势明显 |
| **环境影响** | 尾矿 | 水资源+占地 | 水资源 |
| **技术成熟度** | 成熟 | 成熟 | 发展中 |

## 五、工艺选择决策树

```
Q1: 矿床类型？
├─ 硬岩型 → 锂辉石路线（主流）或锂云母路线
└─ 盐湖型 → Q2

Q2: Mg/Li 比？
├─ <10 → 传统蒸发法（成本最低）
├─ 10-20 → DLE 优先（周期短、回收率高）
└─ >20 → DLE 必需（传统蒸发法不经济）

Q3: 产品需求？
├─ 氢氧化锂 → 硬岩型（盐湖型技术难度大）
└─ 碳酸锂 → 硬岩型或盐湖型均可

Q4: 建设周期要求？
├─ 快速投产（<2年） → 硬岩型或盐湖型DLE
└─ 可接受长周期 → 盐湖型传统蒸发法（成本最低）
```

## 七、项目评估使用指南

### 7.1 硬岩型项目评估

**关键参数**：
1. Li₂O 品位 > 1.0%（经济门槛）
2. 资源量 > 0.5 Mt LCE（规模门槛）
3. 基础设施（电力、水、交通）
4. 政治稳定性

**盈亏平衡**：
- Tier 1：$10,000-15,000/t LCE
- Tier 2：$12,000-18,000/t LCE

### 7.2 盐湖型项目评估

**关键参数**：
1. Li 浓度 > 200 mg/L（经济门槛）
2. **Mg/Li 比**（最关键）：<10 优质，10-20 中等，>20 困难
3. 资源量 > 1 Mt LCE（规模门槛）
4. 气候条件（干旱气候，蒸发量 >2000 mm/年）

**盈亏平衡**：
- Mg/Li < 10：$6,000-10,000/t LCE（传统蒸发法）
- Mg/Li 10-20：$10,000-15,000/t LCE（DLE）
- Mg/Li > 20：$15,000-22,000/t LCE（DLE）

### 7.3 工艺选择建议

| 项目特征 | 推荐工艺 | 理由 |
|---------|---------|------|
| 硬岩型，Li₂O > 1.5% | 锂辉石路线 | 技术成熟，可产氢氧化锂 |
| 盐湖型，Mg/Li < 10 | 传统蒸发法 | 成本最低 |
| 盐湖型，Mg/Li 10-20 | DLE | 周期短，回收率高 |
| 盐湖型，Mg/Li > 20 | DLE | 传统蒸发法不经济 |
| 需要氢氧化锂 | 硬岩型 | 盐湖型技术难度大 |
| 快速投产 | 硬岩型或DLE | 建设周期短 |
