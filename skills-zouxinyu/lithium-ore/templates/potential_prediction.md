## 2. 评价方法学

### 2.1 总体方法
**矿床模型 + 证据权重 (mineral systems + weights of evidence)**：
1. 选定主攻矿床类型 → 写出 mineral systems 关键���素清单
2. 把每要素表达为可制图的"证据层"
3. 按知识 / 数据驱动赋权
4. 逻辑斯谛回归 / WofE / 模糊逻辑 / 随机森林等叠加
5. 输出每网格 0–100 潜力分

### 2.2 工作流程
```
区域类型选定（地质调查比对）
    ↓
模型构建（ knowledge/deposit_types + tectonic_controls 引用）
    ↓
证据层制备（GIS / 化探 / 物探 / 遥感）
    ↓
权重赋值（专家 + 已知矿点训练）
    ↓
潜力分布图生成
    ↓
靶区分级与圈定
    ↓
野外 / 钻探建议
```

### 2.3 数据准备清单
| 数据类型 | 推荐分辨率 | 用途 |
|---------|----------|------|
| 区域地质图 | 1:50,000–1:250,000 | 构造单元、地层分布、花岗岩类侵入体边界 |
| DEM | 30 m 或更高 | 地貌、坡度、构造分析 |
| 航磁 | 100–250 m 线距 | 磁性体识别、深部构造 |
| 重力 | 网格 1–2 km | 密度变化、深部构造 |
| 区域化探 (1:200,000 水系沉积物) | 4 km² 网格 | 元素异常（Li、Ta、Nb、Sn、Be、Rb、Cs） |
| 地化省 (lithogeochemistry) | 不规则 | 成矿背景 |
| 遥感 (Landsat/ASTER/Sentinel-2) | 10–30 m | 蚀变映射（云母化、绿泥石化）/ 盐湖识别 |
| 气候数据 | 网格 1 km | 干旱气候带（盐湖型） |
| 已知矿点位置 | 点数据 | 训练 / 验证 |

### 4.2 概念成矿模型图
{画图：金属源—流体通道—沉淀机制—保存条件}

**硬岩型模型**：花岗岩浆分异 → 残余熔体 → 伟晶岩脉 → 锂辉石/锂云母结晶
**盐湖型模型**：火山岩风化 → 地表水/地下水淋滤 → 封闭盆地汇聚 → 蒸发浓缩 → 卤水层
**粘土型模型**：火山灰沉积 → 热液蚀变 → 粘土矿物吸附 Li

## 5. 证据层制备与权重

### 5.1 证据层清单（举例）
逐层说明每层的：原始数据、处理方法、二值化或连续值、专家阈值。

### 5.2 权重计算方法
- **数据驱动**（首选）：用已知矿点作为正样本，logistic regression / random forest 赋权
- **知识驱动**（兜底）：用专家 AHP（层次分析法）

### 5.3 不确定性来源
- 数据覆盖不均
- 已知矿点偏差（探勘历史富集，不一定代表真实分布）
- 模型外推局限

## 8. 下一步工作建议

### 8.1 短期（3–6 个月）
**硬岩型 A 级靶区**：
- 伟晶岩脉追索（地质填图）
- 槽探 / 浅钻（50–200 m）
- 化学分析（Li₂O、Ta₂O₅、Nb₂O₅、SnO₂、BeO）
- 矿物学分析（锂辉石、锂云母、透锂长石）

**盐湖型 A 级靶区**：
- 卤水取样（钻孔 / 浅井）
- 化学分析（Li、Mg、K、Na、B、SO₄、Cl）
- 蒸发试验（蒸发速率、Mg/Li 比）
- 卤水体积估算（物探 + 钻孔）

**B 级靶区**：地表加密化探 (50×100 m) + 路线地质 + 槽探

### 8.2 中期（6–18 个月）
- A 级靶区资源量初步估算
- B 级靶区按需启动钻探
- 冶炼工艺小试

### 8.3 长期（>18 个月）
- A 级靶区进入预可行 / 可行性研究
- 区域勘查模型迭代

## 9. 主要风险与应对

| 风险 | 处置 |
|------|------|
| 模型误判（硬岩型 vs 盐湖型） | 钻探早期及时判定矿化类型 |
| 伟晶岩分带性强、品位不稳定（硬岩型） | 加密采样、三维建模 |
| Mg/Li 比过高（盐湖型） | DLE 工艺评估 |
| 蒸发池建设周期长（盐湖型） | 提前规划、分期建设 |
| 气候风险（降雨，盐湖型） | 蒸发池防雨设计 |
| 矿权 / 政治不稳（南美锂三角） | 双轨备选区 |
| 物探—化探多解性 | 多元证据收敛后才打孔 |
