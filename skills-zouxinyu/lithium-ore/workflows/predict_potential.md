# Workflow C：区域成矿潜力分析与预测

## 步骤

### Step 1：明确范围与主攻类型
- 与用户确认评价范围（经纬度 / 矿权 / 行政区）
- 与用户确认主攻类型（如不确定，先用 Workflow A 思路做"类型适配性分析"）
- **关键决策**：硬岩型 vs 盐湖型 vs 粘土型（决定后续证据层选择）

### Step 2：读取必读知识
- `knowledge/deposit_types.md`（必读，明确目标类型特征）
- `knowledge/exploration_criteria.md`（必读，列出找矿要素）
- `knowledge/global_belts.md`（必读，定位区域）
- `knowledge/processing_routes.md`（必读，冶炼工艺适配）
- `knowledge/grade_tonnage.md`（按需）

### Step 3：构建控矿模型
- 写出"金属源—流体—通道—沉淀—保存"五要素答案
- **硬岩型**：花岗岩浆分异 → 残余熔体 → 伟晶岩脉 → 锂辉石/锂云母结晶 → 构造保存
- **盐湖型**：火山岩风化 → 地表水/地下水淋滤 → 封闭盆地汇聚 → 蒸发浓缩 → 卤水层保存
- **粘土型**：火山灰沉积 → 热液蚀变 → 粘土矿物吸附 Li → 覆盖层保存
- 给出概念成矿模式图（ASCII / SVG / 调用 canvas-design）

### Step 4：列出证据层清单
按 `templates/potential_prediction.md` 第 4–5 节，列出 GIS 证据层 + 权重。

**硬岩型证据层**：
- 花岗岩类侵入体（特别是 S 型花岗岩）
- 伟晶岩脉出露
- 区域构造（造山带、碰撞带）
- 化探异常（Li、Ta、Nb、Sn、Be、Rb、Cs）
- 遥感蚀变（云母化、绿泥石化）
- 已知伟晶岩型矿点空间关联

**盐湖型证据层**：
- 封闭盆地（内流盆地）
- 火山岩分布（流纹岩、凝灰岩）
- 气候带（干旱、半干旱）
- 盐湖分布（遥感识别）
- 地下水 Li 浓度（如有数据）
- 已知盐湖型矿点空间关联

**粘土型证据层**：
- 火山岩/火山灰分布
- 粘土矿物出露
- 热液蚀变带
- 化探异常（Li、Mg）

### Step 5：分析与赋分
- **有 GIS 数据时**：
  - 调用 `anthropic-skills:xlsx` 处理网格数据
  - 用 logistic regression / WofE / fuzzy logic 叠加
  - 输出 0–100 分潜力分布
- **无 GIS 数据时**：
  - 用知识驱动 + 专家判断打分
  - 在区域地质底图上手工 / AI 圈定靶区
  - 给出方法论清单供用户后续 GIS 化

### Step 6：靶区圈定与分级
按 A（≥80）/ B（60–80）/ C（40–60）/ D（<40）分四级。

### Step 7：靶区描述与建议工程
A 级靶区每个详细描述（一节）；B 级靶区列表式描述；C/D 级仅清单。

**硬岩型建议工程**：
- 伟晶岩脉追索（地质填图）
- 槽探 / 浅钻（50–200 m）
- 化学分析（Li₂O、Ta₂O₅、Nb₂O₅、SnO₂、BeO）
- 矿物学分析（锂辉石、锂云母、透锂长石）

**盐湖型建议工程**：
- 卤水取样（钻孔 / 浅井）
- 化学分析（Li、Mg、K、Na、B、SO₄、Cl）
- 蒸发试验（蒸发速率、Mg/Li 比）
- 卤水体积估算（物探 + 钻孔）

**粘土型建议工程**：
- 粘土层取样
- 化学分析（Li、Mg）
- 矿物学分析（蒙脱石、伊利石、高岭石）
- 提锂工艺小试

### Step 8：资源量量级估算
按目标类型典型品位—吨位 + 靶区面积/深度做概念估算（明确为推断 speculative）。

**硬岩型**：伟晶岩脉体积 × 锂辉石含量 × Li₂O 品位
**盐湖型**：卤水体积 × Li 浓度
**粘土型**：粘土层体积 × Li 品位

### Step 9：输出
- 主报告：`anthropic-skills:docx` 生成 .docx
- 打分表：`anthropic-skills:xlsx` 生成 .xlsx（如有数据）
- 模式图：`anthropic-skills:canvas-design` 或简化 SVG 图
- 文件名：`Lithium_Potential_<区域>_<YYYYMMDD>.docx` + `.xlsx`
- 保存至桌面

## 关键提醒

- **不要凭空编造矿点位置**：靶区需有证据支撑（构造 / 异常 / 类比）
- **明确不确定性**：尤其无 GIS 数据时，所有分级是相对 / 概念性的
- **硬岩型特别注意**：伟晶岩分带性强、锂辉石品位不稳定
- **盐湖型特别注意**：Mg/Li 比决定提锂成本、气候风险（降雨）、蒸发池建设周期长
- **粘土型特别注意**：提锂工艺尚不成熟、经济性不确定
- **不替代实测验证**：所有 A 级靶区都需要钻孔验证；本 skill 仅提供"哪里去钻"的优先级建议

## 与 paper skill 的衔接

潜力分析报告产出后，可作为 paper skill 的素材撰写"区域成矿规律与靶区预测"主题论文。
