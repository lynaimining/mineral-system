# 金矿技术 / 勘查 / 专题报告模板（模式 B 输出）

> 根据用户需求，本模板包含四种子格式。Claude 应先与用户确认子格式，再按对应章节生成 .docx。

---

## 子格式 1：金矿专题地质报告（默认）

适用：单一矿床/区带的成矿规律研究、综合成矿地质研究报告。

### 章节结构
```
封面与责任页
摘要 / Abstract（中英对照）
1  绪论
   1.1  研究背景与意义
   1.2  研究区位置与交通
   1.3  以往工作程度
   1.4  研究方法与工作量
2  区域成矿地质背景
   2.1  大地构造背景
   2.2  区域地层
   2.3  区域岩浆活动
   2.4  区域构造
   2.5  区域成矿规律
3  矿床地质特征
   3.1  地层与沉积相
   3.2  侵入岩 / 沉积建造 / 火山建造
   3.3  构造
   3.4  矿体特征（形态、规模、品位、产状、分布）
   3.5  矿石特征（矿物组合、结构构造、品位变化、nugget effect）
   3.6  围岩蚀变与分带
4  矿床地球化学
   4.1  成矿元素地球化学
   4.2  同位素地球化学（S、Pb、H-O、C-O、Sr-Nd-Hf）
   4.3  流体包裹体
   4.4  黄铁矿微量元素与勘查指示
5  成矿年代学
   5.1  容矿岩浆 / 沉积时代
   5.2  成矿时代（Ar-Ar、Re-Os、SHRIMP）
6  矿床成因
   6.1  成矿物质来源
   6.2  成矿流体来源与演化
   6.3  成矿动力学背景
   6.4  矿床类型厘定
   6.5  成矿模式
7  成矿规律与找矿标志
   7.1  时空分布规律
   7.2  控矿要素
   7.3  找矿标志（地质 + 地化 + 物探 + 遥感）
   7.4  深部 / 外围找矿方向
8  结论与展望
9  参考文献
10 附录（数据表、剖面图、电子文件清单）
```

## 子格式 2：勘查报告（中国 DZ/T 0033 规范）

适用：中国矿权区的预查 / 普查 / 详查 / 勘探报告。

### 章节结构（按规范）
```
1  绪论
   1.1  目的任务
   1.2  位置交通
   1.3  自然地理与经济
   1.4  以往地质工作评述
   1.5  本次工作概况与工作量
2  区域地质
   2.1  地层
   2.2  构造
   2.3  岩浆岩
   2.4  变质作用
   2.5  区域矿产
3  矿区地质
   3.1  地层
   3.2  构造
   3.3  岩浆岩
   3.4  围岩蚀变
   3.5  地球物理 / 地球化学特征
4  矿体（层）地质
   4.1  矿体特征（编号、形态、规模、产状、品位、厚度）
   4.2  矿石质量（矿物组成、化学成分、有用 / 有害组分）
   4.3  矿石加工技术性能（重选、浮选、氰化、难选冶性）
   4.4  矿床开采技术条件（水文、工程、环境地质）
5  勘查工作及质量评述
   5.1  勘查类型与勘查工程间距
   5.2  地形测量
   5.3  地质填图
   5.4  探矿工程（槽探 / 钻探 / 坑探）
   5.5  物探 / 化探
   5.6  采样与化验（nugget effect 处理）
   5.7  原始资料与综合资料
6  资源 / 储量估算
   6.1  工业指标
   6.2  估算方法
   6.3  矿块圈定
   6.4  资源 / 储量分类与估算结果
   6.5  资源 / 储量变化原因分析
7  矿床开发经济意义概略研究（在详查及以上阶段）
8  结论
9  附图、附表、附件
```

## 子格式 3：NI 43-101 / JORC 摘要章节

适用：海外公司公告所需技术报告。

### 章节结构（NI 43-101 标准 22 项）
```
1  Summary
2  Introduction
3  Reliance on Other Experts
4  Property Description and Location
5  Accessibility, Climate, Local Resources, Infrastructure, Physiography
6  History
7  Geological Setting and Mineralization
8  Deposit Types
9  Exploration
10 Drilling
11 Sample Preparation, Analyses, and Security
12 Data Verification
13 Mineral Processing and Metallurgical Testing
14 Mineral Resource Estimates
15 Mineral Reserve Estimates （Pre-Feasibility / Feasibility 阶段）
16 Mining Methods （Pre-FS / FS）
17 Recovery Methods （Pre-FS / FS）
18 Project Infrastructure （Pre-FS / FS）
19 Market Studies and Contracts （Pre-FS / FS）
20 Environmental Studies, Permitting, and Social or Community Impact
21 Capital and Operating Costs （Pre-FS / FS）
22 Economic Analysis （Pre-FS / FS）
23 Adjacent Properties
24 Other Relevant Data and Information
25 Interpretation and Conclusions
26 Recommendations
27 References
28 Date and Signature Page
```

JORC 2012 类似但章节略不同：Public Report 关键章节按 JORC Table 1 三段式（Sampling Techniques and Data / Reporting of Exploration Results / Estimation and Reporting of Mineral Resources / Reserves）。

## 子格式 4：年报 / 市场—地质综合章节

适用：全球金矿市场年度回顾与综合性文档。

### 章节结构
```
1  全球金市场年度回顾
   1.1  价格走势
   1.2  供需平衡
   1.3  央行购金与投资需求
2  全球金矿生产年度回顾（按矿种 / 国家 / 公司）
3  上游—中游—下游产业链动态
   3.1  矿山新建 / 扩产 / 减产
   3.2  冶炼与回收
   3.3  下游需求结构变化（珠宝、投资、工业、央行）
4  全球金资源—储量动态
   4.1  新发现与并购
   4.2  关键产区的政治—矿权事件
5  战略矿带专题（每年聚焦 1–2 个）
   5.1  例：Carlin Trend（微细浸染型金矿）
   5.2  例：Witwatersrand Basin（古砾岩型金矿）
6  长期展望（5–10 年）
   6.1  需求模型
   6.2  供应缺口估算
   6.3  回收与替代影响
7  对中国 / 公司的策略建议
8  附表与附图
```

## 撰写规范（与 `paper` skill 一致）

- **图文并茂**：每节至少 1 张图 / 表，避免大段纯文字。
- **数据可追溯**：所有数字必须注明来源（公司年报、政府部门、SCI 文献、USGS）。
- **引用格式**：
  - 中文报告：GB/T 7714-2015
  - 英文报告：APA / Harvard / SEG style 任选
- **图件类型**（必备）：区域地质 + 矿区地质平面 + 典型剖面 + 蚀变模式 + 钻孔柱状 / 历史—展望趋势图
- **审慎用语**：成因部分用 "Suggests / consistent with / preferred interpretation"，避免绝对化。
- **不确定性披露**：资源量分类、置信区间、关键假设独立小节、nugget effect 影响评估。
- **CrossRef / OpenAlex 验证**：参考文献列表通过 `ref-verify` skill 验证（避免 AI 幻觉）。
