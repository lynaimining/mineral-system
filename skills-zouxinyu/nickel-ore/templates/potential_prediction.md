## 2. 评价方法学

### 2.1 总体方法
**矿床模型 + 证据权重 (mineral systems + weights of evidence)**：
1. 选定主攻矿床类型 → 写出 mineral systems 关键要素清单
2. 把每要素表达为可制图的"证据层"
3. 按知识 / 数据驱动赋权
4. 逻辑斯谛回归 / WofE / 模糊逻辑 / 随机森林等叠加
5. 输出每网格 0–100 潜力分

### 2.2 工作流程
```
区域类型选定（地质调查比对）
    ↓
模型构建（ knowledge/deposit_types + tectonic_controls 引用）
    ↓
证据层制备（GIS / 化探 / 物探 / 遥感）
    ↓
权重赋值（专家 + 已知矿点训练）
    ↓
潜力分布图生成
    ↓
靶区分级与圈定
    ↓
野外 / 钻探建议
```

### 2.3 数据准备清单
| 数据类型 | 推荐分辨率 | 用途 |
|---------|----------|------|
| 区域地质图 | 1:50,000–1:250,000 | 构造单元、地层分布、超镁铁质岩体边界 |
| DEM | 30 m 或更高 | 地貌、坡度、构造分析 |
| 航磁 | 100–250 m 线距 | 磁性体识别、深部构造 |
| 重力 | 网格 1–2 km | 密度变化、深部构造 |
| 区域化探 (1:200,000 水系沉积物) | 4 km² 网格 | 元素异常（Ni、Co、Cr、Cu、PGE） |
| 地化省 (lithogeochemistry) | 不规则 | 成矿背景 |
| 遥感 (Landsat/ASTER/Sentinel-2) | 10–30 m | 蚀变映射（褐铁矿化、蛇纹石化） |
| 气候数据 | 网格 1 km | 热带气候带（红土型） |
| 已知矿点位置 | 点数据 | 训练 / 验证 |

### 4.2 概念成矿模型图
{画图：金属源—流体通道—沉淀机制—保存条件}

**红土型模型**：超镁铁质基岩 → 热带气候风化 → Ni 淋滤富集 → 褐铁矿层/蛇纹石层
**岩浆型模型**：地幔柱/裂谷 → 镁铁质岩浆 → 硫化物熔离 → 重力沉降 → Ni-Cu-PGE 富集

**红土型证据层**：

## 5. 证据层制备与权重

### 5.1 证据层清单（举例）
逐层说明每层的：原始数据、处理方法、二值化或连续值、专家阈值。

### 5.2 权重计算方法
- **数据驱动**（首选）：用已知矿点作为正样本，logistic regression / random forest 赋权
- **知识驱动**（兜底）：用专家 AHP（层次分析法）

### 5.3 不确定性来源
- 数据覆盖不均
- 已知矿点偏差（探勘历史富集，不一定代表真实分布）
- 模型外推局限

## 8. 下一步工作建议

### 8.1 短期（3–6 个月）
**红土型 A 级靶区**：
- 浅钻（10–30 m）网格加密
- 化学分析（Ni、Co、Fe、Mg、Si、Al）
- 矿物学分析（针铁矿、褐铁矿、蛇纹石）
- 冶炼工艺小试

**岩浆型 A 级靶区**：
- 深钻（300–1500 m）验证
- 岩心编录（岩浆分异序列）
- 硫化物矿物鉴定
- PGE 分析

**B 级靶区**：地表加密化探 (50×100 m) + 路线地质 + 槽探

### 8.2 中期（6–18 个月）
- A 级靶区资源量初步估算
- B 级靶区按需启动钻探

### 8.3 长期（>18 个月）
- A 级靶区进入预可行 / 可行性研究
- 区域勘查模型迭代

## 9. 主要风险与应对

| 风险 | 处置 |
|------|------|
| 模型误判（红土型 vs 岩浆型） | 钻探早期及时判定矿化类型 |
| 剥采比过高（红土型） | 经济性评估、优化开采方案 |
| 深部验证成本高（岩浆型） | 物探先行、优化钻孔位置 |
| 冶炼工艺不匹配（红土型） | 小试验证 HPAL vs RKEF |
| 矿权 / 政治不稳（印尼/菲律宾） | 双轨备选区 |
| 物探—化探多解性 | 多元证据收敛后才打孔 |
