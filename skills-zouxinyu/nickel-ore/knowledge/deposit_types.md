## 1. Laterite Ni（红土型镍矿）

### 诊断特征
- **构造背景**：热带-亚热带气候带，超镁铁岩（蛇纹岩/橄榄岩）风化壳
- **容矿岩石**：蛇纹岩、橄榄岩风化产物
- **垂向分带**（从上到下）：
  1. **褐铁矿层（Limonite Zone）**：Fe 富集，Ni 1.0-1.5%，低 Mg
  2. **腐泥土层（Saprolite Zone）**：Mg 富集，Ni 1.5-2.5%，高 Mg，保留原岩结构
  3. **蛇纹岩层（Serpentinite Bedrock）**：未风化基岩，Ni 0.2-0.3%
- **矿物组合**：针铁矿（褐铁矿层）、蛇纹石-绿泥石-滑石（腐泥土层）
- **关键参数**：
  - **Ni/Fe 比**：决定冶炼工艺（高 Ni/Fe → HPAL；低 Ni/Fe → RKEF）
  - **Mg/Fe 比**：腐泥土层高 Mg（>20%），褐铁矿层低 Mg（<5%）
- **共生元素**：Co（0.05-0.15%）、Cr、Mn

### 三大亚类

| 亚类 | 关键特征 | 适用工艺 | 代表矿床 |
|------|---------|---------|---------|
| **Limonite-dominant** | 褐铁矿层为主，低 Mg 高 Fe | RKEF（镍铁） | 印尼 Weda Bay、菲律宾 Rio Tuba |
| **Saprolite-dominant** | 腐泥土层为主，高 Mg 低 Fe | HPAL（硫酸镍） | 新喀里多尼亚 Goro、印尼 Halmahera |
| **Mixed** | 褐铁矿+腐泥土混合 | HPAL 或 RKEF | 菲律宾 Taganito、印尼 Morowali |

### 典型矿床（按规模）
- **Tier 1（>1 Mt Ni）**：
  - **Goro**（新喀里多尼亚，~4.5 Mt Ni @ 1.6%）
  - **Weda Bay**（印尼，~3 Mt Ni @ 1.5%）
  - **Ramu**（巴布亚新几内亚，~1.5 Mt Ni @ 1.5%）
- **Tier 2（0.1-1 Mt Ni）**：
  - 印尼苏拉威西数十个矿床（Morowali、Pomalaa、Sorowako）
  - 菲律宾巴拉望/苏里高数十个矿床

### 全球分布
- **环太平洋红土带**：印尼（全球最大）、菲律宾、新喀里多尼亚、巴布亚新几内亚
- **加勒比海**：古巴（Moa Bay）、多米尼加
- **南美**：巴西、委内瑞拉、哥伦比亚
- **其他**：澳大利亚、马达加斯加

### 关键近年进展
- **印尼镍矿爆发**（2020-2026）：禁止镍矿石出口 → 强制国内冶炼 → RKEF/HPAL 产能激增
- **电池级镍需求**：HPAL 工艺产硫酸镍（Class 1）成为新能源产业链核心
- **环境争议**：红土镍矿开采对热带雨林破坏、HPAL 尾渣处理问题

## 2. Magmatic Ni-Cu-PGE（岩浆硫化物镍铜矿）

### 诊断特征
- **构造背景**：大陆裂谷、地幔柱、大火成岩省（LIP）
- **容矿岩石**：镁铁-超镁铁质侵入体（辉长岩-橄榄岩-纯橄岩）
- **矿物组合**：镍黄铁矿（pentlandite）+ 磁黄铁矿（pyrrhotite）+ 黄铜矿（chalcopyrite）± 黄铁矿
- **硫化物形态**：块状硫化物、浸染状硫化物、网脉状硫化物
- **关键成矿条件**：
  1. 高镁玄武质岩浆（MgO > 10%）
  2. 硫饱和（外来硫源：地壳混染、蒸发岩）
  3. 岩浆通道系统（动态补给）
- **共生元素**：Cu（0.5-2%）、Co（0.05-0.1%）、PGE（Pt+Pd 1-5 g/t）

### 与 copper-ore 的 Cu-Ni-PGE 类型重叠
**共享知识库矿床**：
- Norilsk（俄罗斯，>10 Mt Ni + 15 Mt Cu）
- Voisey's Bay（加拿大，~3 Mt Ni + 1.5 Mt Cu）
- Jinchuan（中国，~5 Mt Ni + 1 Mt Cu）
- Sudbury（加拿大，>10 Mt Ni + 10 Mt Cu，陨石撞击成因）

### 典型矿床（按规模）
- **Tier 0（>5 Mt Ni）**：Norilsk、Sudbury、Jinchuan
- **Tier 1（1-5 Mt Ni）**：Voisey's Bay、Thompson（加拿大）、Pechenga（俄罗斯）
- **Tier 2（0.1-1 Mt Ni）**：Raglan（加拿大）、Eagle（美国）

### 关键文献
- **Naldrett (2004) Econ. Geol. 100th Anniv.** 经典综述
- **Arndt et al. (2005)** 地幔柱与镍矿关系
- **Li et al. (2012)** Jinchuan 成矿模型

## 3. Komatiite-hosted Ni（科马提岩型镍矿）

### 诊断特征
- **构造背景**：���古宙绿岩带（2.7-3.0 Ga）
- **容矿岩石**：科马提岩（超镁铁质火山岩，MgO > 18%）
- **矿物组合**：镍黄铁矿 + 磁黄铁矿 + 黄铜矿
- **硫化物形态**：
  - **Massive sulfide**：块状硫化物，Ni 5-15%
  - **Matrix sulfide**：基质硫化物，Ni 2-5%
  - **Disseminated sulfide**：浸染状硫化物，Ni 0.5-2%
- **关键成矿条件**：
  1. 科马提岩熔岩流底部（basal contact）
  2. 硫饱和（火山喷发过程中硫化）
  3. 地形低洼（熔岩流汇聚）

### 典型矿床
- **Tier 1**：Kambalda（澳大利亚，>1 Mt Ni @ 3-5%）
- **Tier 2**：
  - **澳大利亚 Yilgarn Craton**：数十个矿床（Leinster、Perseverance、Mt Keith）
  - **加拿大 Abitibi**：Alexo、Dundonald
  - **津巴布韦**：Trojan、Shangani

### 关键文献
- **Barnes et al. (2016) Ore Geology Reviews** Komatiite-hosted Ni-Cu-PGE deposits 综述
- **Lesher & Keays (2002)** Kambalda 成矿模型

## 4. Hydrothermal Ni（热液型镍矿）

### 诊断特征
- **构造背景**：海底热液系统、蛇绿岩
- **容矿岩石**：蛇纹岩、玄武岩
- **矿物组合**：针镍矿（millerite）、紫硫镍矿（violarite）、镍黄铁矿
- **成因**：海底热液循环，Ni 从超镁铁岩中淋滤富集

### 典型矿床
- **Oman 蛇绿岩**：小规模热液镍矿
- **现代海底热液**：大西洋中脊、东太平洋海隆（资源潜力，未开采）

---

## 5. Black Shale Ni（黑色页岩型镍矿）

### 诊断特征
- **构造背景**：缺氧盆地，黑色页岩沉积
- **容矿岩石**：富有机质黑色页岩
- **矿物组合**：镍黄铁矿、黄铁矿
- **成因**：海水中 Ni 被有机质吸附富集

### 典型矿床
- **Talvivaara**（芬兰，~1 Mt Ni @ 0.3%，多金属：Ni-Zn-Cu-Co）

## 类型快速鉴别决策树（用于 workflow A）

```
Q1: 是否为风化壳？
├─ 是 → 垂向分带（褐铁矿/腐泥土/蛇纹岩）？
│   ├─ 是 → Laterite Ni
│   └─ 否 → 检查其他类型
└─ 否 → 是否为岩浆硫化物？
    ├─ 是 → 容矿岩石？
    │   ├─ 科马提岩（太古宙） → Komatiite-hosted
    │   └─ 辉长岩-橄榄岩（元古-显生宙） → Magmatic Ni-Cu-PGE
    └─ 否 → 是否为热液/沉积？
        ├─ 海底热液 → Hydrothermal Ni
        └─ 黑色页岩 → Black Shale Ni

Q2: 共生金属"指纹"？
├─ Ni-Co（无 Cu） → Laterite Ni
├─ Ni-Cu-PGE → Magmatic Ni-Cu-PGE / Komatiite-hosted
├─ Ni-Zn-Cu-Co → Black Shale Ni
└─ Ni（单一） → Hydrothermal Ni

Q3: 地质时代？
├─ 太古宙（2.7-3.0 Ga） → Komatiite-hosted
├─ 元古-显生宙 → Magmatic Ni-Cu-PGE
└─ 新生代（风化） → Laterite Ni
```

## 冶炼工艺适配（镍矿特有）

| 矿石类型 | 适用工艺 | 产品 | 产品等级 | 主要应用 |
|---------|---------|------|---------|---------|
| 红土矿（高 Mg 腐泥土） | HPAL（高压酸浸） | 硫酸镍（NiSO₄） | Class 1 | 三元锂电池 |
| 红土矿（低 Mg 褐铁矿） | RKEF（回转窑-电炉） | 镍铁（NPI） | Class 2 | 不锈钢 |
| 红土矿（混合） | Ferronickel（火法） | 镍铁 | Class 2 | 不锈钢 |
| 岩浆硫化物 | Flotation + Smelting | 电解镍 | Class 1 | 电池/合金/电镀 |
| 科马提岩型 | Flotation + Smelting | 电解镍 | Class 1 | 电池/合金/电镀 |

**关键判断参数**：
- **Ni/Fe 比**：>0.015 适合 HPAL，<0.015 适合 RKEF
- **Mg/Fe 比**：>2 适合 HPAL（腐泥土），<1 适合 RKEF（褐铁矿）
- **产品价差**：Class 1（电解镍/硫酸镍）价格 > Class 2（镍铁）价格 20-30%
