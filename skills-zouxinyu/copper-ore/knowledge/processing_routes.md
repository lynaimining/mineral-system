# Copper Ore Processing Routes

## 一、斑岩铜矿标准流程

### 1.1 浮选工艺

**适用矿石**：
- 硫化铜矿（黄铜矿 CuFeS₂、辉铜矿 Cu₂S、斑铜矿 Cu₅FeS₄）
- 品位范围：0.3-2.0% Cu
- 粒度要求：-200 mesh 占 60-80%

**工艺流程**：
```
破碎（三段一闭路）→ 磨矿（球磨+分级）→ 粗选 → 扫选 → 精选（2-3次）→ 铜精矿（20-28% Cu）
```

**药剂制度**：
- 捕收剂：黄药类（乙基黄药、丁基黄药）100-150 g/t
- 起泡剂：松醇油 20-40 g/t
- 调整剂：石灰（pH 10-12）500-1500 g/t
- 抑制剂：硫酸锌+氰化物（抑制黄铁矿）

**技术指标**：
- 粗精矿品位：15-20% Cu
- 最终精矿品位：20-28% Cu
- 回收率：85-92%（取决于矿石性质）
- 尾矿品位：<0.15% Cu

**典型案例**：
- Escondida（智利）：0.55% Cu 原矿 → 28% Cu 精矿，回收率 88%
- Oyu Tolgoi（蒙古）：0.51% Cu 原矿 → 24% Cu 精矿，回收率 86%
- Grasberg（印尼）：0.95% Cu 原矿 → 26% Cu 精矿，回收率 90%

### 1.2 火法冶炼

**闪速熔炼（Flash Smelting）**：
- 铜精矿 + 氧气 → 冰铜（45-75% Cu）+ 炉渣
- 温度：1200-1300°C
- 优势：能耗低、环保、自热反应
- 代表：Outokumpu 工艺（全球 50% 产能）

**转炉吹炼（Converting）**：
- 冰铜 + 氧气 → 粗铜（98-99% Cu）+ SO₂
- 温度：1150-1250°C
- 产物：粗铜（含 Au、Ag、Pt、Pd 等贵金属）

**火法精炼（Fire Refining）**：
- 粗铜 → 阳极铜（99.5% Cu）
- 去除 S、O、Fe 等杂质
- 为电解精炼准备原料

### 1.3 电解精炼

**工艺参数**：
- 电解液：CuSO₄ + H₂SO₄
- 电流密度：200-300 A/m²
- 温度：60-65°C
- 周期：10-28 天

**产品质量**：
- 阴极铜：99.99% Cu（LME A级标准）
- 阳极泥：回收 Au、Ag、Pt、Pd、Se、Te

**经济性**：
- 电耗：250-350 kWh/t 阴极铜
- 贵金属回收价值可抵消 30-50% 精炼成本

### 1.4 经济参数

**资本成本（CAPEX）**：
- 选矿厂（10万吨/日）：$15-25亿
- 冶炼厂（50万吨/年铜精矿）：$8-12亿
- 电解精炼厂（30万吨/年阴极铜）：$3-5亿

**运营成本（OPEX）**：
- 采矿：$8-15/吨原矿
- 选矿：$6-12/吨原矿
- 冶炼：$200-400/吨铜精矿
- 精炼：$150-250/吨阴极铜
- **总现金成本（C1）**：$1.2-2.5/lb Cu（取决于品位和规模）

## 二、SSC 高品位矿直接冶炼

### 2.1 适用条件

**矿石特征**：
- 品位：>5% Cu（部分矿段可达 20-60% Cu）
- 矿物：辉铜矿、铜蓝、自然铜为主
- 杂质：低 As、Sb、Bi（<0.1%）

**经济优势**：
- 跳过浮选环节，直接火法冶炼
- 降低 CAPEX 和 OPEX
- 适合小规模、高品位矿床

### 2.2 工艺优势

**vs 传统浮选**：
- CAPEX 降低 40-60%（无需建大型选矿厂）
- OPEX 降低 30-50%（无浮选药剂、尾矿库成本）
- 建设周期缩短 1-2 年

**适用项目**：
- Kamoa-Kakula（刚果金）：部分矿段 5-10% Cu，直接冶炼
- Kipushi（刚果金）：11% Cu + 8% Zn，直接冶炼

### 2.3 工艺流程

```
破碎 → 筛分（>5% Cu 直接入炉，<5% Cu 浮选）→ 闪速熔炼 → 转炉 → 电解
```

**技术挑战**：
- 需严格控制入炉品位稳定性
- 高品位矿段储量有限，需配矿

## 三、IOCG 多金属矿复杂选冶

### 3.1 Cu-Au-U-REE 分离

**矿石特征**：
- Cu：0.5-2.0%
- Au：0.5-3.0 g/t
- U₃O₈：0.01-0.1%
- REE（LREE为主）：0.1-1.0%

**选矿流程**：
```
磨矿 → Cu 浮选 → Au 重选/浮选 → U 浸出 → REE 浸出
```

**典型案例**：
- Olympic Dam（澳大利亚）：1.2% Cu + 0.5 g/t Au + 0.05% U₃O₈
- Ernest Henry（澳大利亚）：0.9% Cu + 0.6 g/t Au

### 3.2 工艺挑战

**技术难点**：
- Cu-Au 共生，需优化浮选药剂避免 Au 损失
- U 放射性，需专门处理设施和许可
- REE 回收需酸浸，环保压力大

**解决方案**：
- Cu 浮选后，尾矿重选回收粗粒 Au
- U 在冶炼阳极泥中富集，单独提取
- REE 视价格决定是否回收（通常不回收）

### 3.3 经济性

**副产品贡献**：
- Au 贡献 30-50% 总收入
- U 贡献 10-20% 总收入（Olympic Dam）
- REE 通常不回收（成本>收益）

**C1 成本**：
- $0.8-1.5/lb Cu（Au、U 副产品抵扣后）

## 四、VMS 多金属矿联合选冶

### 4.1 Cu-Zn-Pb 分离

**矿石特征**：
- Cu：1-3%
- Zn：3-8%
- Pb：0.5-2%
- Ag：50-150 g/t
- Au：0.5-2 g/t

**选矿流程**：
```
磨矿 → Cu 优先浮选 → Zn 浮选 → Pb 浮选 → 尾矿（含 Au、Ag）
```

### 4.2 工艺流程

**Cu 浮选**：
- pH 10-11，黄药捕收
- 精矿品位：20-25% Cu
- 回收率：85-90%

**Zn 浮选**：
- pH 11-12，黄药捕收
- 精矿品位：50-55% Zn
- 回收率：80-88%

**Pb 浮选**：
- pH 8-9，黄药捕收
- 精矿品位：55-65% Pb
- 回收率：75-85%

### 4.3 副产品回收

**Ag、Au 分布**：
- 70-80% 进入 Cu 精矿
- 15-20% 进入 Pb 精矿
- 5-10% 进入 Zn 精矿

**冶炼回收**：
- Cu 冶炼阳极泥回收 Au、Ag
- Pb 冶炼回收 Ag（主要来源）
- Zn 冶炼回收少量 Ag

**典型案例**：
- Neves-Corvo（葡萄牙）：5% Cu + 2% Zn + 100 g/t Ag
- Kidd Creek（加拿大）：2% Cu + 7% Zn + 1% Pb + 80 g/t Ag

## 五、副产品回收经济性

### 5.1 Mo 回收（Cu-Mo PCD）

**矿石特征**：
- Cu：0.4-0.8%
- Mo：0.01-0.05%（100-500 ppm）

**工艺流程**：
```
Cu-Mo 混合浮选 → Cu-Mo 分离（Na₂S 抑制 Cu）→ Mo 精矿（45-55% Mo）
```

**经济贡献**：
- Mo 价格：$15-25/lb（波动大）
- 贡献总收入：10-25%
- 典型项目：Bingham Canyon（美国）、Chuquicamata（智利）

### 5.2 Au 回收（Cu-Au PCD）

**矿石特征**：
- Cu：0.3-0.6%
- Au：0.2-0.8 g/t

**回收路径**：
- 80-90% Au 进入 Cu 精矿
- 冶炼阳极泥回收（回收率 >95%）

**经济贡献**：
- Au 价格：$1800-2200/oz
- 贡献总收入：15-30%
- 典型项目：Grasberg（印尼）、Cadia（澳大利亚）

### 5.3 Co 回收（Cu-Co SSC）

**矿石特征**：
- Cu：2-5%
- Co：0.1-0.5%

**工艺流程**：
```
Cu 浮选 → 冶炼 → 电解液中 Co 富集 → 萃取分离 → Co 产品
```

**经济贡献**：
- Co 价格：$15-35/lb（波动极大）
- 贡献总收入：10-20%
- 典型项目：Kamoa-Kakula（刚果金）、Tenke Fungurume（刚果金）

### 5.4 Re 回收（Mo-rich PCD）

**矿石特征**：
- Cu：0.4-0.7%
- Mo：0.03-0.08%
- Re：0.5-2 ppm（极低但极贵）

**回收路径**：
- Re 随 Mo 进入精矿
- Mo 焙烧时 Re 挥发，烟气回收
- 产品：高铼酸铵（NH₄ReO₄）

**经济贡献**：
- Re 价格：$1000-3000/kg（全球最贵金属之一）
- 贡献总收入：5-15%
- 典型项目：Collahuasi（智利）、Los Pelambres（智利）

## 六、不同品位对应的最优工艺选择

### 6.1 高品位（>2% Cu）

**推荐工艺**：
- SSC 型：直接冶炼（品位 >5%）或浮选（2-5%）
- PCD 型：常规浮选 + 火法冶炼

**经济特点**：
- C1 成本：$0.8-1.5/lb Cu
- IRR：>25%
- 回收期：3-5 年

### 6.2 中品位（0.5-2% Cu）

**推荐工艺**：
- 浮选 + 火法冶炼（主流）
- 重点优化磨矿细度和药剂制度

**经济特点**：
- C1 成本：$1.2-2.0/lb Cu
- IRR：15-25%
- 回收期：5-8 年

### 6.3 低品位（0.3-0.5% Cu）

**推荐工艺**：
- 大规模露天开采 + 浮选
- 必须有副产品（Au、Mo）支撑经济性

**经济特点**：
- C1 成本：$1.8-2.5/lb Cu
- IRR：10-15%
- 回收期：8-12 年

### 6.4 极低品位（<0.3% Cu）

**推荐工艺**：
- 氧化矿：堆浸-SX-EW
- 硫化矿：生物浸出（研发阶段）

**经济特点**：
- C1 成本：$2.0-3.0/lb Cu
- IRR：<10%（通常不经济）
- 需超大规模（>10 万吨/日）

## 七、氧化铜矿处理

### 7.1 堆浸-萃取-电积（SX-EW）

**适用矿石**：
- 氧化铜矿（孔雀石、蓝铜矿、赤铜矿）
- 品位：0.3-1.5% Cu
- 酸耗：<50 kg H₂SO₄/t 矿石

**工艺流程**：
```
破碎（-50mm）→ 堆浸（H₂SO₄）→ 萃取（LIX 984N）→ 电积 → 阴极铜（99.99% Cu）
```

**技术参数**：
- 堆浸周期：60-180 天
- Cu 回收率：60-80%
- 电积电流密度：250-350 A/m²

### 7.2 经济参数

**CAPEX**：
- 堆浸场（5 万吨/日）：$3-5 亿
- SX-EW 厂（5 万吨/年 Cu）：$1.5-2.5 亿

**OPEX**：
- 采矿：$2-4/吨矿石
- 堆浸：$3-6/吨矿石
- SX-EW：$0.3-0.5/lb Cu
- **总 C1 成本**：$1.5-2.2/lb Cu

### 7.3 环保优势

**vs 浮选+冶炼**：
- 无 SO₂ 排放
- 无尾矿库（堆浸渣可回填）
- 水循环利用率 >90%

**典型案例**：
- Morenci（美国）：0.25% Cu 氧化矿，SX-EW
- Radomiro Tomic（智利）：0.6% Cu 氧化矿，SX-EW

## 八、生物浸出技术

### 8.1 适用矿石

**目标矿石**：
- 难选硫化铜矿（辉铜矿、铜蓝）
- 低品位硫化矿（<0.3% Cu）
- 浮选尾矿（0.1-0.2% Cu）

**微生物**：
- 嗜酸氧化亚铁硫杆菌（Acidithiobacillus ferrooxidans）
- 嗜酸氧化硫硫杆菌（Acidithiobacillus thiooxidans）

### 8.2 工艺流程

**堆浸生物浸出**：
```
破碎 → 造堆 → 接种细菌 → 喷淋 H₂SO₄ → 浸出液 → SX-EW → 阴极铜
```

**搅拌槽生物浸出**：
```
磨矿 → 浸出槽（细菌+氧气）→ 固液分离 → SX-EW → 阴极铜
```

### 8.3 经济性

**技术指标**：
- 浸出周期：堆浸 180-365 天，搅拌槽 3-7 天
- Cu 回收率：堆浸 50-70%，搅拌槽 80-90%
- 温度：30-45°C（中温菌）

**成本**：
- OPEX：$1.8-2.5/lb Cu（高于传统浮选）
- 适合处理尾矿和低品位矿

**商业案例**：
- Escondida（智利）：硫化矿生物堆浸，3 万吨/年 Cu
- Cerro Verde（秘鲁）：硫化矿生物堆浸，6 万吨/年 Cu

## 九、2026 年技术趋势

### 9.1 绿色冶炼技术

**氢气闪速熔炼**：
- H₂ 替代部分焦炭，降低碳排放
- 试点：Boliden（瑞典）、Glencore（加拿大）

**电炉熔炼**：
- 100% 电力驱动，零碳排放
- 成本：需低价绿电（<$0.03/kWh）

**直接电解**：
- 铜精矿直接电解，跳过火法冶炼
- 研发阶段，尚未商业化

### 9.2 数字化与自动化

**智能选矿**：
- 在线品位分析（XRF、LIBS）
- AI 优化药剂添加
- 案例：Rio Tinto Kennecott（美国）

**自动化冶炼**：
- 无人值守转炉
- 实时炉况监控
- 案例：Codelco Ventanas（智利）

### 9.3 循环经济

**废杂铜回收**：
- 2026 年全球 30% Cu 来自再生铜
- 电解精炼处理废杂铜

**尾矿再选**：
- 老尾矿库再选回收 Cu（0.1-0.2%）
- 案例：Bingham Canyon 尾矿再选项目

## 十、工艺选择决策树

```
矿石类型？
├─ 氧化矿
│  ├─ 品位 >0.5% → 堆浸-SX-EW（首选）
│  └─ 品位 <0.5% → 大规模堆浸或放弃
│
└─ 硫化矿
   ├─ 品位 >5% → 直接冶炼（SSC 型）
   ├─ 品位 2-5% → 浮选 + 火法冶炼
   ├─ 品位 0.5-2% → 浮选 + 火法冶炼（需副产品）
   ├─ 品位 0.3-0.5% → 大规模浮选（需副产品+规模效应）
   └─ 品位 <0.3% → 生物浸出（研发）或放弃
```

**副产品考量**：
- 有 Au/Mo → 品位下限可降至 0.3%
- 有 Co → 品位下限可降至 0.4%（SSC 型）
- 无副产品 → 品位下限 0.5%

## 十一、项目评估使用指南

### 11.1 工艺选择评估

**Step 1：确定矿石类型**
- 氧化矿 vs 硫化矿 vs 混合矿
- 查阅钻孔岩芯氧化率数据

**Step 2：确定品位区间**
- 平均品位、可采品位、边界品位
- 不同品位对应不同工艺

**Step 3：识别副产品**
- Au、Mo、Co、Ag 等
- 副产品可显著改善经济性

**Step 4：选择工艺路线**
- 参考本文档工艺选择决策树
- 对比 CAPEX、OPEX、回收率

### 11.2 回收率估算

**浮选回收率**：
- 高品位（>1% Cu）：88-92%
- 中品位（0.5-1% Cu）：85-88%
- 低品位（<0.5% Cu）：80-85%

**堆浸回收率**：
- 氧化矿：70-80%
- 混合矿：60-70%
- 硫化矿（生物浸出）：50-70%

**冶炼回收率**：
- 火法冶炼：95-98%
- SX-EW：90-95%

### 11.3 成本估算

**快速估算公式**：
```
C1 成本（$/lb Cu）= [采矿成本 + 选矿成本 + 冶炼成本 - 副产品收入] / Cu 产量（lb）
```

**参考值（2026 年）**：
- 大型 PCD（>0.5% Cu）：$1.2-1.8/lb
- 中型 PCD（0.3-0.5% Cu）：$1.8-2.3/lb
- SSC（>2% Cu）：$0.8-1.5/lb
- 氧化矿 SX-EW：$1.5-2.2/lb

**敏感性分析**：
- 品位 ±10% → C1 成本 ±15-20%
- 回收率 ±5% → C1 成本 ±8-12%
- 副产品价格 ±20% → C1 成本 ±5-15%
