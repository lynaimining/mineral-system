# 各类型铜矿勘查要素与靶区识别

> 来源：Sillitoe (2010) Porphyry Copper Systems（"圣经"级 vector 综述）；MDRU/CODES 工业指南；用户文件夹内 `2019-State-of-the-art analysis of geochemical data for mineral exploration.pdf`、`2024-driving mineral system knowledge using multi-modal, collocated, scale-consistent characterisation.pdf`。

## 通用思路：成矿系统四问

撰写或评价任何铜矿靶区时，先回答四个问题（Wyborn et al. mineral systems 框架）：
1. **金属源**（Cu 从哪来？）—— 岩浆？地壳？围岩？
2. **流体源**（运 Cu 的载体？）—— 岩浆水、盆地卤水、变质水？
3. **运移通道**（Cu 怎么走过来的？）—— 断层、不整合、岩性界面、岩浆通道？
4. **沉淀机制**（Cu 为什么停下来？）—— 降温、降压、沸腾、还原、流体混合？

## 1. 斑岩 Cu（PCD）勘查要素

### 1.1 区域 / 矿带尺度（百千米）
- 弧岩浆带的弧弯—弧后转换段
- 大型走滑—断裂系统切弧带（如 Domeyko 断裂系）
- 大火山区域 + 副火山岩相
- 板片几何：俯冲角变陡 / 平板段 / 板片撕裂边缘
- 时代：与 100 Myr 以内的弧岩浆活动相关

### 1.2 矿床尺度（千米—十千米）
- 多期次斑岩侵入复合岩体（早—中—晚）
- 斑岩岩体小（<2 km²）但脉动多
- **蚀变分带（Sillitoe model 垂向）**：
  ```
  顶部（剥蚀露头标志）：
   ├─ 高级泥化 lithocap（叶蜡石—明矾石—硅化）
   ├─ 高硫型浅成低温 vein 系统
  中部（矿体）：
   ├─ 绢英岩化（quartz-sericite-pyrite）+ D 脉
   ├─ 钾化（K-fsp + biotite + magnetite）+ A/B 脉 ← 主矿体
  外带 / 围岩：
   ├─ 青磐岩化（chlorite-epidote-calcite）
  深部：
   └─ 钠—钙蚀变 + 深部斑岩根
  ```
- 蚀变投影规模 1–5 km；矿化中心通常 0.5–2 km
- 角砾岩筒（breccia pipe）—— 早期 + 同矿期 + 晚期类型

### 1.3 地化标志
- 土壤 / 岩屑 Cu-Mo-Au 复合异常
- Au/Cu 比变化：碱性 PCD 高、钙碱性 PCD 中、钼斑岩低
- 路径元素：W、Bi、Te、Se（高温）；As、Sb、Hg、Tl（顶盖）
- 同位素：Cu 同位素稍重（δ⁶⁵Cu > 0），Mo 同位素稍重（与流体演化相关）
- **锆石微量元素（高分辨筛选）**：Ce⁴⁺/Ce³⁺ > 300、Eu/Eu* > 0.4、(Eu/Eu*)/Y × 10⁴ > 1.0

### 1.4 物探标志
- 磁化率：核部钾化磁铁矿强磁异常（高） + 外带绢英岩化弱磁（低）形成"环带"
- IP/Resistivity：黄铁矿外带高极化率（chargeability）+ 中心硅化低电阻
- Gravity：常有局部高重力（含矿斑岩比围岩稍致密 + 钾化致密化）
- AMT/MT：深部熔体 / 流体响应

### 1.5 遥感
- 短波红外 (SWIR/ASTER)：明矾石、叶蜡石、白云母、绿泥石映射
- 高光谱 (Hyperion, EnMAP)：氧化铁 + 黏土综合带

### 1.6 找矿"得分要点"（每项是/否赋分）
- [ ] 区域是否在 known PCD 矿带 ±100 km
- [ ] 是否有同期中酸性斑岩侵入体出露
- [ ] 是否见到典型蚀变分带（钾化—绢英岩化—青磐岩化）
- [ ] 是否有 Cu-Mo-Au 化探异常（>背景 5×）
- [ ] 是否发育石英脉（A/B/D 脉序列）
- [ ] 是否见到角砾岩筒
- [ ] 锆石筛选指标合格
- [ ] 物探（磁、IP）有同心异常
- [ ] 遥感蚀变带有规模（>1 km²）
- [ ] 没有过度剥蚀（lithocap 顶板已侵蚀）的负面证据

## 2. SSC 勘查要素

### 2.1 区域尺度
- **盆地几何**：裂谷期—被动陆缘转换；裂谷半地堑可识别
- **基底**：稳定克拉通或地块
- **沉积充填**：底部红色砂岩/砾岩 + 上覆细粒红 / 黑 / 白互层 + 蒸发岩 / 黑色页岩
- **时代**：元古宙—古生代为主

### 2.2 盆地内部尺度
- **氧化—还原界面**：Roan 群 R3 段（中非）；Kupfer 黑页岩（中欧）；Nonesuch shale（白松，美国）
- **构造圈闭**：盆缘断层、盐构造、不整合面
- **白云岩化**：高 Mg 卤水通过证据
- **铜矿化层位（marker bed）**：在 known belt 中追索

### 2.3 矿化标志
- 自下而上：赤铁矿—蓝辉铜—辉铜—斑铜—黄铜—黄铁矿"分带"
- "Roll-front"几何（卷状矿体，地下水活动产物）
- 后期改造：脉状矿化沿断层穿层
- 富 Co：高 Co/Cu（>0.05）—— 中非铜带特色

### 2.4 物探—化探
- 土壤 Cu-Co±Ag 异常 + 路径 Pb-Zn-Mo
- AMT / TEM 探测氧化—还原界面（电阻率反差）
- 重力探测盆地基底深度
- 地震 (2D) 在覆盖区识别盆地几何

### 2.5 中非铜带专属指南（用户重点）
- **赞比亚 Lower Roan = R2.1.3 = 矿化层**
- **DRC Mines Group = 矿层位**
- 表层富集：DRC 浅层 Cu-Co 氧化—硅酸盐矿（水钴矿、孔雀石、硅孔雀石）
- Lumwana 型：与造山改造高度相关（脉—剪切带矿化）
- Kamoa-Kakula 型：保存极完好的同生—早成岩 Cu 富集 + 后期再活化

## 3. IOCG 勘查要素

### 3.1 区域尺度
- 元古宙稳定克拉通
- A 型花岗岩—双峰式岩浆事件
- 大型走滑—伸展系统
- 蒸发岩盖层（提供 Cl 卤水）

### 3.2 矿床尺度
- 角砾岩筒 / 角砾岩席 / 隐爆角砾
- 多期蚀变叠加：钠—钙 → 钾化 → 赤铁矿化 → 硅化—碳酸盐化
- 铁氧化物体（磁铁矿—赤铁矿）数十米到千米尺度
- 矿物组合：Cu-Au-U-REE-F-P—— 可作为"金属指纹"（与 PCD 区分关键）
- 缺失典型黄铁矿 +/- 钼

### 3.3 物探（IOCG 是物探"显眼"的类型）
- **高重力 + 高磁同时异常**——经典 IOCG 标志
- 电极化率：低—中等（黄铁矿少）
- 放射性：U 异常（沥青铀矿）

### 3.4 找矿要点
- [ ] 元古宙克拉通内或边缘
- [ ] 同期 A 型花岗岩 / 双峰岩浆
- [ ] 钠—钙蚀变（钠长石—阳起石—磁铁矿）发育
- [ ] 高重力 + 高磁同步异常
- [ ] 钻孔见铁氧化物角砾岩 + Cu-Au-U-REE 矿化
- [ ] 没有典型 PCD 蚀变（钾化—绢英岩化—青磐岩化）

## 4. VMS 勘查要素

### 4.1 区域
- 海相弧—弧后扩张中心
- 双峰式火山岩（玄武岩—流纹岩）
- 半石化沉积盖层（古海底)

### 4.2 矿床
- 块状硫化物透镜体 + 下伏蚀变筒
- 蚀变筒：内带绿泥石—硅化、外带绢云母—青磐岩化
- 矿物分带：底 Cu-rich → 上部 Zn-Pb-rich（"双层"模型）
- 化学沉积铁锰帽（Mn-Fe gossan, jasper, exhalite）—— 重要找矿标志

### 4.3 物探—化探
- 电磁法（VTEM/AeroTEM/SQUID-TEM）—— VMS 主流找矿手段
- 重力小高、磁中—高（含磁黄铁矿）
- 自然电位（SP）异常
- 化探：Cu-Zn-Pb-Au-Ag-Ba-Hg 复合
- "Exhalite"（含铁碧玉、燧石、黑矿、白矿层）—— 古海底沉积标志层

## 5. 岩浆 Cu-Ni-PGE 勘查要素

### 5.1 区域
- 大火成岩省（LIP）/ 古地幔柱
- 镁铁—超镁铁岩通道（dunite, peridotite, gabbro, pyroxenite）
- 围岩：含硫沉积岩（蒸发岩、黑色页岩、煤层）—— 提供外源 S

### 5.2 矿床
- 通道型：基底凹陷、岩浆"漏斗"、岩流细颈处
- Reef 型：层状岩浆房特定层位（Bushveld Merensky 距底部 ~2 km）
- 矿物：磁黄铁矿 + 镍黄铁矿 + 黄铜矿 + 五硫四金刚组合 + PGM
- Ni-Cu 比：通道型 ~1:1，Reef 型偏 PGE

### 5.3 找矿要点
- [ ] 区域有 LIP / 大型镁铁—超镁铁岩出露
- [ ] 围岩含 S（蒸发岩、煤、含黄铁矿黑色页岩）
- [ ] 见到岩浆 sulfide（半侵入式硫化物或 net-textured ore）
- [ ] 高强度磁异常 + 中重力
- [ ] EM / SQUID-TEM 异常（强导体）
- [ ] 化探 Ni-Cu-PGE 复合 + 路径 As-Bi-Te-Se-Co

## 6. Skarn Cu 勘查要素

### 6.1 区域
- 弧—碰撞带，与 PCD 共生（"PCD-skarn 系统"）
- 容矿岩：碳酸盐岩（灰岩、白云岩）
- 同期中酸性侵入体

### 6.2 矿床
- 内带（侵入体侧）：石榴—辉石—硅化
- 外带（碳酸盐侧）：硅灰石—透辉石—绿帘石
- "矽卡岩"接触带宽 1–500 m
- 块状磁铁矿—黄铜矿—斑铜矿层
- 后期角砾化 + 浅成低温叠加（Cu-Au-Te 加富）

### 6.3 找矿要点
- 卫星 PCD 系统的远程矽卡岩
- 化探 Cu-Mo-Zn-Au 复合
- 重磁高 + IP 中等

## 7. 浅成低温热液 Cu-Au（与 PCD 系统共生）

### 7.1 类型
- 高硫型 (HS)：明矾石—叶蜡石—硅帽，Cu-Au
- 低硫型 (LS)：冰长石—绢云母，Au-Ag

### 7.2 找矿要点
- HS 与下伏 PCD 直接关联，是"剥蚀深度浅"指示
- 蚀变筒可保留至浅地表，遥感易识别

## 数据—证据—置信度的勘查决策原则

1. **多源证据收敛**：单一异常不立项；至少 ≥3 类独立证据（地质 + 化探 + 物探）收敛才打钻。
2. **分层 + 分级**：靶区分 A/B/C 级，对应不同投资节奏。
3. **风险提示**：
   - "地表见矿"≠"深部经济矿体"——警惕表层富集（supergene）误导
   - 成矿模型 ≠ 真矿——模型是推论，钻孔是验证
   - 同位素 + 年代学 ≠ 经济性——可论证成因不能保证规模
4. **现代趋势**（用户文件夹内 2024 论文）：多模态—多尺度—一致性表征 + 机器学习辅助靶区勾画。
