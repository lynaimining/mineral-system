1. **造山型金矿带区块**（如有Mahd adh Dhahab/Al Amar延伸区）

2. **IOCG潜力区块**（如有铁氧化物蚀变+Cu-Au异常）

## 实战找矿要素清单

### 造山型金矿（Orogenic Gold）- 沙特最重要类型

#### 必备要素（5个，缺一不可）
1. ✅ **构造**：NE-SW剪切带（Najd断裂系统，640-620 Ma）
2. ✅ **蚀变**：碳酸盐化-绢云母化-硅化（肉眼可见白色碳酸盐脉）
3. ✅ **岩性**：绿片岩相变质火山-沉积岩
4. ✅ **地化**：Au-As-Sb-W异常（As/Au比值10-100）
5. ✅ **地球物理**：磁低（黄铁矿化）+ 电阻率高（硅化）

#### 钻孔部署原则
```
优先级1：剪切带 + 碳酸盐化 + Au异常（>50 ppb）
    → 孔深：200-400m
    → 孔距：50m（首批验证）
    → 方位：垂直剪切带走向

优先级2：剪切带 + 蚀变 + 无化探数据
    → 先做土壤化探（100m×25m网格）
    → 再根据异常布孔

优先级3：剪切带 + 无蚀变
    → 低优先级，可能已剥蚀
```

#### 典型矿床参数（类比用）
| 矿床 | 资源量 | 品位 | 矿体特征 | 找矿标志 |
|------|--------|------|----------|----------|
| Mahd adh Dhahab | >1 Moz | 10-20 g/t | 石英-碳酸盐脉，宽1-5m | 强碳酸盐化，Au-As-Sb |
| Al Amar | ~2 Moz | 2-3 g/t | 蚀变带型，宽10-50m | 绢云母化，Au-As |
| Bulghah | 0.5 Moz | 5-8 g/t | 高品位石英脉，宽0.5-2m | 强硅化，可见金 |

#### 成本估算（快速）
- 土壤化探：$50-100/样（含分析）
- 地面磁法：$200-500/km²
- 钻探：$150-250/m（沙特成本）
- 首批验证（3孔×300m）：~$150,000

### VMS Cu-Zn-Au（火山块状硫化物）- 铜金共生

#### 必备要素
1. ✅ **岩性**：双峰式火山岩（玄武岩+流纹岩）
2. ✅ **层位**：海相沉积盖层（黑色页岩/硅质岩）
3. ✅ **蚀变**：绿泥石化-绢云母化（足迹蚀变）
4. ✅ **地化**：Cu-Zn-Au-Ag异常（Cu/Zn比值0.5-2）
5. ✅ **地球物理**：磁高（磁黄铁矿）+ 激电异常（块状硫化物）

#### 钻孔部署原则
```
优先级1：磁异常 + 激电异常 + Cu-Zn异常
    → 孔深：300-600m（测试块状硫化物层位）
    → 孔距：100m（首批）
    → 方位：垂直火山岩层理

优先级2：流纹岩顶部 + 黑色页岩接触带
    → 这是VMS最可能产出位置
    → 即使无异常也值得打1-2个验证孔

优先级3：足迹蚀变带（绿泥石化）
    → 沿蚀变带向上追索块状硫化物
```

#### 典型矿床参数（类比用）
| 矿床 | 资源量 | 品位 | 矿体特征 | 找矿标志 |
|------|--------|------|----------|----------|
| Jabal Sayid | 12 Mt | 2.2% Cu, 0.7 g/t Au | 块状硫化物，厚5-20m | 强磁异常，Cu-Zn-Au |
| Al Masane | 5 Mt | 1.8% Cu, 1.2% Zn | 透镜状，厚3-15m | 激电异常，Cu-Zn |

#### 成本估算
- 航磁+航重：$50-100/km²
- 激电测量：$500-1000/km
- 钻探：$200-300/m（更深更贵）
- 首批验证（2孔×500m）：~$250,000

### IOCG Cu-Au-U（铁氧化物-铜-金）- 高风险高回报

#### 必备要素（沙特尚未发现大型IOCG，但条件具备）
1. ✅ **构造**：元古宙克拉通边缘，深大断裂
2. ✅ **蚀变**：钠-钙-铁氧化物蚀变（赤铁矿-磁铁矿）
3. ✅ **地化**：Cu-Au-U-REE异常（多元素组合）
4. ✅ **地球物理**：大型磁异常（>1000 nT）+ 重力高
5. ✅ **岩性**：镁铁-超镁铁岩或火山岩

#### 钻孔部署原则
```
优先级1：磁高 + 重力高 + Cu-Au异常中心
    → 孔深：500-1000m（IOCG通常较深）
    → 孔距：200m（首批，IOCG规模大）
    → 方位：垂直磁异常长轴

警告：IOCG勘探风险高，建议：
    1. 先做详细物探（航磁+航重+ZTEM）
    2. 再做地面磁法+激电验证
    3. 确认靶区后再钻探
```

#### 全球类比（沙特无案例，参考其他地区）
| 矿床 | 资源量 | 品位 | 特征 |

| Olympic Dam（澳大利亚） | >10 Gt | 1.2% Cu, 0.5 g/t Au, 0.03% U | 超大型，深埋 |
| Candelaria（智利） | 500 Mt | 1% Cu, 0.2 g/t Au | 大型磁异常 |

#### 成本估算
- 航磁+航重+ZTEM：$150-300/km²
- 深孔钻探：$300-500/m
- 首批验证（2孔×800m）：~$600,000

## 投资决策矩阵

### 买矿决策树（3个关键问题）

```
Q1: 成矿类型确定吗？
    ├─ 造山型金矿（有剪切带+蚀变+Au异常）
    │   → 确定性高，可以买
    │   → 估值：$50-150/oz Au（资源量）
    │
    ├─ VMS（有双峰式火山岩+Cu-Zn异常）
    │   → 确定性中，需验证钻探
    │   → 估值：$0.3-0.8/lb Cu（资源量）
    │
    └─ IOCG（只有磁异常+Cu异常）
        → 确定性低，高风险
        → 估值：大幅折扣或不买

Q2: 有多大？（Tier分级）
    ├─ Tier 1（>100 t Au 或 >5 Mt Cu）
    │   → 必买，战略资产
    │   → 溢价20-50%
    │
    ├─ Tier 2（10-100 t Au 或 1-5 Mt Cu）
    │   → 可买，主力项目
    │   → 市场价
    │
    └─ Tier 3（<10 t Au 或 <1 Mt Cu）
        → 谨慎，除非高品位
        → 折价30-50%

Q3: 基础设施如何？
    ├─ 距离公路<50km，有电有水
    │   → 加分，CAPEX低
    │
    ├─ 距离公路50-150km
    │   → 中性，需建设
    │
    └─ 距离公路>150km，无水无电
        → 减分，CAPEX高
        → 资源量需更大才经济
```

### 快速估值公式

#### 金矿（资源量阶段）
```
估值 = 资源量（oz） × 金价（$/oz） × 折扣系数

折扣系数：
- Measured（331）：0.3-0.5
- Indicated（332）：0.2-0.3
- Inferred（333）：0.1-0.15

示例：
10 Moz @ 3 g/t（Indicated）
= 10M oz × $2,350/oz × 0.25
= $5.9B × 0.25
= $1.5B 估值
```

#### 铜矿（资源量阶段）
```
估值 = 资源量（lb Cu） × 铜价（$/lb） × 折扣系数

折扣系数：
- Measured：0.15-0.25
- Indicated：0.10-0.15
- Inferred：0.05-0.10

示例：
5 Mt @ 2% Cu（Indicated）
= 5M t × 2% × 2204.6 lb/t
= 220M lb Cu
= 220M × $4.2/lb × 0.12
= $111M 估值
```

## 风险提示

### 地质风险（按概率排序）
1. **剥蚀风险（60%概率）**
   - ANS经历多期抬升，浅部矿体可能已剥蚀
   - 缓解：选择低剥蚀区（盆地边缘、断陷带）

2. **构造破坏（40%概率）**
   - 多期变形导致矿体破碎
   - 缓解：选择构造相对简单区

3. **蚀变叠加（30%概率）**
   - 晚期蚀变掩盖成矿蚀变
   - 缓解：详细蚀变矿物学研究

### 经济风险
1. **金价波动**：当前$2,350/oz，盈亏平衡约$1,200/oz
2. **铜价波动**：当前$4.2/lb，盈亏平衡约$2.5/lb
3. **CAPEX超支**：沙漠环境，水电成本高

### 政策风险
1. **矿权稳定性**：沙特政策相对稳定
2. **环保要求**：日益严格
3. **本地化要求**：需雇佣沙特员工
