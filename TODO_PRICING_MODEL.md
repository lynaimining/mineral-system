## 下次会话待办（优先级排序）

### P0：补充缺失因子数据（最关键）

逐笔WebSearch补充以下数据：
1. **品位（g/t Au）** — 每笔交易的项目品位
2. **AISC（$/oz）** — 每笔交易的全维持成本
3. **开采方式细分** — 露天/地下/混合，补充深度和剥采比
4. **矿山寿命（LOM）** — 剩余矿山寿命
5. **基建成熟度** — 已建成/在建/绿地

### P1：扩充样本量到50+笔

搜索方向：
- 2013-2015年的金矿交易（当前数据空白）
- 中国国内金矿交易（紫金、山东黄金、中金等）
- 小型交易（<$500M，更多resource/exploration阶段）
- 澳大利亚ASX上的小型金矿并购

### P2：改进模型

1. **换用随机森林** — 非线性，更适合小样本多因子
2. **添加交互项** — stage × gold_price（高金价时production溢价更大）
3. **添加定性调整因子** — 基建溢价、竞标溢价、战略溢价
4. **目标R²≥0.70**

### P3：定性调整机制

在模型预测基础上，叠加定性因子调整：

```python
adjusted_ev = model_ev * (1
    + grade_premium      # 品位溢价：高品位(>5g/t)+15-25%，低品位(<2g/t)-10-20%
    + infra_premium      # 基建溢价：已建成+10-20%，绿地-20-30%
    + bidding_premium    # 竞标溢价：多买家+20-40%
    + strategic_premium  # 战略溢价：补充pipeline+15-30%
    + control_premium    # 控制权溢价：100%收购+20-30%
    - political_discount # 政治折价：高风险国家-20-40%
)
```

### P4：验证与回测

- 用2-3个已知交易做留一验证（leave-one-out）
- 对比模型预测 vs 实际成交价
- 计算预测误差的分布

## 关键发现（已验证）

1. **阶段是最强预测因子** — 每升一级+$24/oz
2. **金价对EV/oz的直接影响弱** — 因为买卖双方都看到同样的金价
3. **规模效应显著** — 大矿比小矿贵（非线性）
4. **国别差异大** — 发达国家溢价，非洲/南美折价
5. **地下矿略有溢价** — 可能因为品位更高
